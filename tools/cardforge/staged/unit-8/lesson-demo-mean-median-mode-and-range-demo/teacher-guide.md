# Teacher Guide — Mean, Median, Mode, and Range

> CardForge package · **DEMO / sample** · status: **draft**

## Overview
Students learn four ways to summarize a set of numbers — mean, median, mode, and range — and decide which one best describes a typical value, especially when an outlier is present.

## Objective & Standard
- **Objective:** I can find the mean, median, mode, and range of a data set.
- **Language objective:** I can explain which measure I used with the words mean, median, mode, range, and outlier.
- **Standard:** 6.SP.3
- **Time:** ~45 min

## Materials
- Pencil and practice page
- Optional: calculator
- Optional: sticky notes for ordering data

## Warm-Up
Put these numbers in order from least to greatest: 7, 2, 9, 4, 2. (Answer: 2, 2, 4, 7, 9.) Which number appears most often?

## Key Vocabulary (ESOL-friendly)
- **Mean** _(es: Media)_ — The average. Add all the numbers, then divide by how many there are. _(El promedio. Suma todos los números y divide entre cuántos hay.)_
  - Example: Mean of 2, 4, 6 → (2+4+6) ÷ 3 = 4
- **Median** _(es: Mediana)_ — The middle number when the data is in order. _(El número del medio cuando los datos están en orden.)_
  - Example: 1, 3, 5, 7, 9 → median = 5
- **Mode** _(es: Moda)_ — The number that appears most often. _(El número que aparece más veces.)_
  - Example: 2, 3, 3, 5 → mode = 3
- **Range** _(es: Rango)_ — The biggest number minus the smallest number. _(El número mayor menos el número menor.)_
  - Example: 4, 9, 20, 5 → range = 20 − 4 = 16
- **Outlier** _(es: Valor atípico)_ — A number much bigger or smaller than the rest. _(Un número mucho mayor o menor que los demás.)_
  - Example: 10, 12, 11, 50 → 50 is an outlier

## Formulas / Definitions
- mean = (sum of all values) ÷ (how many values)
- median = middle value in order (if two middles, average them)
- mode = value that appears most often (can be none or more than one)
- range = greatest value − least value

## Modeled Examples (I do)
**Modeled Example 1.** Find the mean, median, mode, and range of the test scores: 80, 85, 90, 85, 70.

   1. Order the data: 70, 80, 85, 85, 90.
   2. Mean: 70 + 80 + 85 + 85 + 90 = 410, then 410 ÷ 5 = 82.
   3. Median: the middle of five numbers is the 3rd one → 85.
   4. Mode: 85 appears twice (most often) → 85.
   5. Range: 90 − 70 = 20.

   **Answer:** Mean = 82, Median = 85, Mode = 85, Range = 20.

**Modeled Example 2.** Find the mean, median, mode, and range of reading minutes: 10, 12, 15, 18.

   1. Order the data: 10, 12, 15, 18 (already in order).
   2. Mean: 10 + 12 + 15 + 18 = 55, then 55 ÷ 4 = 13.75.
   3. Median: with four numbers, average the two middles → (12 + 15) ÷ 2 = 13.5.
   4. Mode: every number appears once → there is no mode.
   5. Range: 18 − 10 = 8.

   **Answer:** Mean = 13.75, Median = 13.5, Mode = none, Range = 8.

## Guided Practice (We do)
1. Find the mean of 4, 8, 6, 2, 5.
   - **Answer:** Mean = 5 (sum 25 ÷ 5).
2. Find the median of 3, 9, 1, 7, 5.
   - **Answer:** Order 1,3,5,7,9 → median = 5.

## Independent Practice (You do)
1. Find the mean of 4, 8, 6, 2, 5.
2. Find the median of 3, 9, 1, 7, 5.
3. Find the mode of 2, 4, 4, 6, 4, 8.
4. Find the range of 12, 7, 20, 5, 16.
5. Test scores: 88, 92, 100, 88, 76, 96. Find the mean, median, mode, and range.
6. Daily steps: 5000, 7000, 6000, 8000, 6000. Find the mean, median, mode, and range.
7. For 10, 12, 14, 16, 18, 100, would the mean or the median better describe a typical value? Explain.
8. Create your own data set of 5 numbers that has a mode of 9 and a range of 10.

## Common Misconceptions
- **Finding the median without ordering the numbers first.** → Always rewrite the data least-to-greatest before picking the middle.
- **Thinking every data set has exactly one mode.** → A set can have no mode (all different) or more than one mode.
- **Confusing range (a spread) with the largest value.** → Range is greatest minus least, not just the biggest number.

## Supports
- **ESOL:** Bilingual EN/ES vocabulary box with worked examples; Sentence frame: "The ___ is ___ because ___."; Color-code or underline the middle number when finding the median
- **SPED:** Provide the data already ordered for the first few problems; Reduce to problems 1–4 plus one multi-part item; Allow a calculator for the mean
