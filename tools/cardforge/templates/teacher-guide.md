# Template — Teacher Guide (reference)

Rendered by `lib/generate.mjs → renderTeacherGuide(job)` from the `lesson`
block of a job file. Sections produced, in order:

1. Overview
2. Objective & Standard (objective, language objective, standard, time)
3. Materials
4. Pacing table (phase / time / what)
5. Warm-Up
6. Key Vocabulary (ESOL-friendly, EN/ES with examples)
7. Formulas / Definitions
8. Modeled Examples — I do (numbered steps + answer)
9. Guided Practice — We do (with answers)
10. Independent Practice — You do (prompts only)
11. Answer Key pointer (`answer-key.md`)
12. Common Misconceptions (misconception → fix)
13. Supports (ESOL; SPED)
14. Exit Ticket pointer (`exit-ticket.md`)
15. Extension
16. Substitute / Emergency Notes

This is documentation only; edit the renderer, not this file, to change output.
