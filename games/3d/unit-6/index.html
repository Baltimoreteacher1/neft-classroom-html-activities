Expression Engine — Build the Expression

Select a number, variable (x), or operation block (← / → or tap), drop it into the engine (Space or tap), and assemble an expression that hits the target value — or is equivalent for every x. Submit with Enter or ↑. Remove the last block with ↓. The engine evaluates live using order of operations.
