Run-of-show

  • 0:59–1:00 Transition. Naming overreliance is easy; redesigning is the work. Today we fix an AI support that went too far.
  • 1:00–1:02 Launch. Chat: “Open the HTML, find your group tab, read the scenario first, then redesign the AI move.” Remove the barrier without doing the student’s job.
  • 1:02–1:06 Model. Don’t just make the prompt nicer — change the scaffold so students return to the source/task and do the thinking.
  • 1:06–1:15 Group work — 9-min timer. Chat: “Post only when your group can defend aim, structure, and evidence.” Visit rooms or monitor chat.
  • 1:15–1:18 Gallery reflection. Scan two posts, borrow one redesign move, and name what changed: aim, structure, evidence, or tool choice.

Chat moves

  • Drop only the prompt participants need right now.
  • Read 3–4 responses aloud, then move on.
  • Keep the timer visible for all group work.

Every support must pass the same check before it posts: Barrier removed · Thinking preserved · Learning target unchanged.
