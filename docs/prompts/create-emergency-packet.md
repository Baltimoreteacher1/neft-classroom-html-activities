# Create an Emergency Sub Packet

Produce a stand-alone, 3-day substitute packet for a lesson — printable, black
-and-white, runs itself.

The sub packet is generated by the Factory from a lesson `job.json`
(`lib/sub-packet.mjs`). To create or refresh one:

1. Make sure the lesson has a `job.json` (see `/generate-lesson-bundle`). The sub
   packet uses its `warmUp`, `modeledExamples`, `practice` (+ `answerKey`), and
   `extension`.
2. Build: `npm run cardforge:build -- <job.json>` → produces `sub-packet.html`
   in the staged dir.
3. Open `sub-packet.html` and check, in this order:
   - Cover page with simple sub directions.
   - 3 day pages, each with a warm-up, a worked example, and 3–4 practice
     problems with work space.
   - A Challenge page for early finishers.
   - An Answer Key page, labeled teacher-only, on its own page.
4. Confirm it is B/W-friendly, large readable font, no tiny text, no
   color-dependent instructions, no filler. Print-preview to verify pagination.

If anything is thin (e.g. fewer than 9 practice problems for 3 days), add more
practice to the `job.json` and rebuild.
