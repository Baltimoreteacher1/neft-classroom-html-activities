# Create an Interactive Activity

Produce a self-contained interactive practice page students run on a Chromebook.

Generated by the Factory from a lesson `job.json` (`lib/interactive.mjs`). It
includes: name entry (no PIN, no teacher dashboard), clear directions, one
problem at a time, a hint after a wrong answer, a stronger hint after 2 tries,
a progress bar, and a print-your-result button.

Steps:

1. Ensure the lesson `job.json` has `practice` problems with `answer` (and `work`
   for the strong hint). Items with `teacherJudgment: true` are skipped (no
   single right answer to auto-check).
2. Build: `npm run cardforge:build -- <job.json>` → `interactive.html`.
3. Smoke-test in a browser: enter a name → start → answer one wrong (hint
   appears) → answer correct (advances) → finish (score + print). Confirm 0
   console errors.
4. Check the language is ESOL-friendly (short sentences, plain words) and the
   page works without any network (all CSS/JS inline).

Do not add a PIN, login, or teacher dashboard unless I explicitly ask.
