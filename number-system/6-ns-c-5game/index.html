📚 Vocabulary Foundations
Match terms to definitions, then test with multiple choice. Key concepts bolded throughout.
Positive numbers are greater than zero (right of 0 on number line). Negative numbers are less than zero (left of 0). Zero is the reference point — neither positive nor negative.
Integer
A whole number that includes positive numbers, negative numbers, and zero (e.g., ..., −3, −2, −1, 0, 1, 2, 3, ...).
Positive Number
A number greater than zero. Often written with a + sign, but the + is optional (e.g., +5 or just 5).
Negative Number
A number less than zero. Always written with a − sign (e.g., −7).
Zero
The origin on the number line. Represents no change, balance, or sea level. Neither positive nor negative.
