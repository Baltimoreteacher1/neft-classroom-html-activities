🗺 What is 6.NS.C.8?
The Big Idea — navigating all four quadrants and measuring distance
📌 Standard
Solve real-world and mathematical problems by graphing points in all four quadrants of the coordinate plane. Use coordinates and absolute value to find distances between points with the same first or second coordinate.
Distance = |x₁ − x₂| (same y) or |y₁ − y₂| (same x)
📐 The coordinate plane has 4 quadrants, an x-axis, a y-axis, and an origin (0,0). Every point is an ordered pair (x, y).
📏 Distance rule: Points on the same vertical line → subtract y-values. Points on the same horizontal line → subtract x-values. Always take absolute value!
⚠️ 6.NS.C.8 only requires distance for points sharing an x or y coordinate (horizontal/vertical lines). No diagonal distance (Pythagorean theorem is Grade 8).

📏 Distance via Absolute Value — The Core Idea
Why absolute value? Because distance is always positive — direction doesn't matter.
Same x-coordinate → Vertical Distance
Points A(−3, 4) and B(−3, −2) share x = −3. They're on the same vertical line.
d = |4 − (−2)| = |4 + 2| = |6| = 6
Same y-coordinate → Horizontal Distance
Points C(2, 5) and D(7, 5) share y = 5. They're on the same horizontal line.
d = |2 − 7| = |−5| = 5

🪜 How to Find Distance — Step by Step
Two methods: same x-coordinate (vertical) and same y-coordinate (horizontal)
1
Identify the two points
Write both ordered pairs clearly. Label them Point A and Point B.
A(−3, 4) and B(−3, −2)
2
Check: do they share an x-coord or y-coord?
Same x-value → vertical distance (subtract y's). Same y-value → horizontal distance (subtract x's).
Both have x = −3 → vertical → subtract y-values
3
Subtract the coordinates that differ
Take one value minus the other. It doesn't matter which order — absolute value fixes it.
4 − (−2) = 4 + 2 = 6
4
Apply absolute value
Distance is always positive — wrap the result in | |. This handles cases where you subtract in the "wrong" order.
|4 − (−2)| = |6| = 6 ✓
5
Label your answer with units
State the distance with units: "6 units." In real-world problems, use the actual unit (miles, feet, blocks).
Distance = 6 units
📘 Three Worked Examples
Vertical distance · Horizontal distance · Real-world context
📍 VERTICAL DISTANCE (same x)
Find the distance between P(2, 7) and Q(2, −3).
→
Same x-coordinate (x = 2) → vertical → subtract y-values
→
|7 − (−3)| = |7 + 3| = |10| = 10
→
Check: P is at y = 7 (above) and Q at y = −3 (below) → 7 units up + 3 units down = 10 ✓
Distance = 10 units
📍 HORIZONTAL DISTANCE (same y)
Find the distance between M(−5, 3) and N(2, 3).
→
Same y-coordinate (y = 3) → horizontal → subtract x-values
→
|−5 − 2| = |−7| = 7
→
Or: |2 − (−5)| = |7| = 7 — same result!
Distance = 7 units
🏙️ REAL-WORLD (city grid)
On a city map, a library is at (−4, 2) and a park is at (3, 2). One unit = 1 block. How many blocks apart?
→
Same y-coordinate (y = 2) → horizontal distance
→
|−4 − 3| = |−7| = 7
7 blocks apart

⚠️ Common Mistakes — Watch Out!
Real student errors and corrections
❌
Mistake #1 — Reversing x and y when plotting
Error: Plotting (3, −4) by going down 3 and right 4 instead of right 3 and down 4.

Fix: Ordered pair is always (x, y) — x is horizontal (left/right) FIRST, then y is vertical (up/down).
❌
Mistake #2 — Subtracting the wrong pair
Error: For same x-coordinate, student subtracts x-values instead of y-values.

Fix: Same x → they're on a vertical line → subtract the values that DIFFER (the y-values).
❌
Mistake #3 — Forgetting to use absolute value
Error: Getting a negative distance. e.g., −5 − 2 = −7, then writing "distance = −7 units."

Fix: Distance is always positive. |−7| = 7. Always take the absolute value after subtracting.
❌
Mistake #4 — Subtracting across quadrants incorrectly with negatives
Error: |4 − (−2)| → student computes as |4 − 2| = 2 instead of |4 + 2| = 6.

Fix: Subtracting a negative = adding its positive. 4 − (−2) = 4 + 2 = 6.
❌
Mistake #5 — Confusing quadrant numbers
Error: Thinking Quadrant I is upper left or numbering them clockwise.

Fix: Start Q I in the upper RIGHT (+,+). Go counterclockwise: II upper left (−,+), III lower left (−,−), IV lower right (+,−).
