⚙️ The Expression Engine WebQuest

Expressions — feed the blocks into the machine and build math!

1. Introduction

Welcome to the Expression Engine! ⚙️ This machine turns numbers, letters, and signs into expressions.

An expression is a math phrase like 3 + x or 2 · n. It has no equal sign. A variable is a letter that stands for a number, like x or n.

In this WebQuest, you will read, write, and build expressions to run the engine.

2. Task

By the end, you will be able to:

  • Write exponents (like 4³ = 4·4·4).
  • Turn words into an expression (like "5 more than n" = n + 5).
  • Evaluate an expression by plugging in a number.
  • Spot equivalent expressions (two that are equal).

3. Process

Do the steps in order. Read each one carefully.

  1. Step 1 — Learn the words

    Variable — a letter for an unknown number. Example: x.
    Coefficient — the number multiplied by a variable. In 5x, the 5 is the coefficient.
    Exponent — tells how many times to multiply a number by itself. In 4³, the 3 is the exponent.
    Evaluate — put a number in for the variable and find the value.

  2. Step 2 — Play the 3D game

    Open the Expression Engine 3D game. Feed blocks into the machine to build expressions.

  3. Step 3 — Read exponents

    4³ means 4 · 4 · 4 = 64. The exponent 3 counts how many 4s you multiply.

    5² = 5 · 5 = 25

  4. Step 4 — Words to expression

    "3 more than n" becomes n + 3. "Twice a number x" becomes 2x.

  5. Step 5 — Evaluate

    Evaluate 2x + 1 when x = 4: 2·4 + 1 = 9.

  6. Step 6 — Show what you know

    Type your name in the bar above, finish the quiz, then Save as PDF or DOC to turn it in.

💡 Tip: 5x means 5 times x. A number next to a letter always means multiply.
