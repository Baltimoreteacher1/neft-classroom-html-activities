Paste & run — step by step

  1. Click Copy code above (or Download .gs).
  2. Open script.google.com → New project.
  3. Delete the starter function myFunction() {} and paste the copied code.
  4. Press Run (▶). Choose buildDoNowForm if prompted.
  5. Click Review permissions → choose your Google account → Allow (first run only).
  6. Open Execution log (View → Logs). The new Google Form's edit URL is printed there.

What each output is

Google Apps Script (.gs) — used here

A short program that runs inside Google's own environment (script.google.com). When you press Run, Google's FormApp service creates the Form in your Drive and prints its URL. This site never sees your Google account, so no login is needed here. This is the route this tool uses.

True one-click creation — not implemented

Creating a Google Form directly from this page (one click, no paste) would require Google OAuth and the Forms API running on a backend server, plus you signing in. That is intentionally not built here to keep the tool login-free, static, and student-data-free. The Apps Script route gives the same result with one paste and one Run.

Teacher tool only. No student data is collected or stored. Your draft questions stay in this browser (localStorage) and never leave your device.
