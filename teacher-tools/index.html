Content rules

How this professional hub stays clean as it grows

Every future card should follow the same structure so the page remains usable instead of becoming a random link collection.

Plain name

Use the tool’s real name, then explain the practical job it does in one sentence.

Task family

Label cards by user goal: command, data, instruction, curriculum, or deployment.

Parallel copy

Descriptions should answer what it is, what it helps with, and who should use it.

Unique action

CTA text must say exactly where the card goes, not repeat generic “Open activity.”
