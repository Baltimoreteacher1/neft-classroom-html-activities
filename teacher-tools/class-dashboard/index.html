What this is

A read-only, at-a-glance gradebook for the results your lessons save automatically. It reads the rma_gradebook array from this browser's localStorage and summarizes it three ways: a gradebook grid, a per-student card, and a per-standard class view.

Privacy

Everything is local-first. Nothing leaves this browser — no network calls, no accounts. Data lives on this device, in this browser only. For student privacy, prefer student IDs or initials over full names when students type their name into a lesson.
