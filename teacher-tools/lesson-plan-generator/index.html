Generation Mode

Default mode composes the plan entirely in your browser (no network, no keys). Optional AI mode is a clearly-marked stub: it is disabled unless you supply an endpoint and falls back to local generation. See the NOTES block at the top of app.js.

No API key is stored or shipped with this tool. If you use AI mode, route through your own proxy that holds the key server-side.
