# HyperDoc Spec (one per unit)  — Path: /hyperdocs/unit-N/index.html

A HyperDoc following Engage → Explore → Explain → Apply → Reflect. Self-contained.
- THEME + KIT: same includes (neft-theme + nt-activity-kit), --nt-* vars, body
  class="nt-theme-aware".
- SECTIONS: Engage (hook/question), Explore (links to the unit 3D game /games/3d/
  unit-N/, math hub, novels), Explain (short plain explanation + worked example),
  Apply (an auto-graded set via NTKit.grade({activityId:"hyperdoc-unit-N", ...})),
  Reflect (1-2 short response prompts captured into the saved doc).
- Save-as-PDF / Save-as-DOC with student name (from NTKit) so completed HyperDocs
  upload to a grading folder. Content matched to THIS unit's standard. Plain
  Grade-6-ESOL language, large readable text, AA contrast, keyboard accessible.
