Warm-up

New data set, in order: 3, 7, 7, 10, 13. First, what is the mean? Add all five, then divide by 5.

Teaching notes

Key idea: Mean = add then divide, median = the middle of the ordered numbers, mode = the value that shows up most.

I do:

  1. My small data set: 2, 4, 4, 6, 9. There are 5 numbers.
  2. Mean: add them, 2 + 4 + 4 + 6 + 9 = 25, then divide by 5. So the mean = 25 ÷ 5 = 5.
  3. Median: the numbers are already in order. The middle (3rd) value is 4, so the median = 4.
  4. Mode: the number 4 appears twice, more than any other, so the mode = 4.

You do: Next, you will order the team's game scores and find the mean, median, and mode yourself. Then you will decide which one best shows a typical game.

Common misconception

A common mistake in Mean, Median, and Mode is skipping the key idea: "Mean = add then divide, median = the middle of the ordered numbers, mode = the value that shows up most." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Mean, Median, Mode, Outlier, Data distribution, Variability (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.
