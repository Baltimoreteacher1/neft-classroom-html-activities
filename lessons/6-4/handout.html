Properties of Operations

6.EE.3 · Unit 6 · Lesson 4

Name   Period

Today's Goal / Meta de hoy

I can use the commutative, associative, and identity properties to rewrite expressions.

Key Idea / Idea clave

Changing the order or the grouping of numbers being added or multiplied does not change the answer.

I Notice / Observo

  • Does the order you add the musicians change the total?
  • What if you group different sections together first — does the total change?

I Wonder / Me pregunto

  • Do these same rules work for subtraction or division?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Commutative Property
Propiedad conmutativa		You can change the order and get the same answer.
Puedes cambiar el orden y obtener la misma respuesta.
Associative Property
Propiedad asociativa		You can change the grouping and get the same answer.
Puedes cambiar la agrupación y obtener la misma respuesta.
Identity Property
Propiedad de identidad		Adding 0 or multiplying by 1 keeps the same value.
Sumar 0 o multiplicar por 1 mantiene el mismo valor.
Property
Propiedad		A rule that is always true in math.
Una regla que siempre es verdadera en matemáticas.
Coefficient
Coeficiente		The number in front of a letter, like the 3 in 3x.
El número frente a una letra, como el 3 en 3x.
Like terms
Términos semejantes		Terms with the same letter, like 2x and 5x.
Términos con la misma letra, como 2x y 5x.
