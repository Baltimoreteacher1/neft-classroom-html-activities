Unit 1, Lesson 5: Add and Subtract Decimals6.NS.3

A quick guide for families and for any student who missed class.

What students are learning

A decimal is a number with a point that shows parts smaller than one whole. To add or subtract decimals, you line up the decimal points so the same place values match.

Key idea: Line up the decimal points, annex zeros if needed, then add or subtract one column at a time.

Key words

  • Decimal — A number with a dot that shows a part between whole numbers.
    Español: Decimal — Un número con un punto que muestra una parte entre números enteros.
  • Place value — What a digit is worth based on where it sits in a number.
    Español: Valor posicional — Lo que vale un dígito según el lugar que ocupa en el número.
  • Tenths — The first spot after the decimal point. It shows parts out of 10.
    Español: Décimas — El primer lugar después del punto decimal. Muestra partes de 10.
  • Hundredths — The second spot after the decimal point. It shows parts out of 100.
    Español: Centésimas — El segundo lugar después del punto decimal. Muestra partes de 100.
  • Annex zeros — Adding zeros at the end of a decimal so numbers line up evenly.
    Español: Agregar ceros — Agregar ceros al final de un decimal para que los números se alineen.

Example

  1. The tank has 128.75 liters. A shuttle adds 46.8 liters. I line up the decimal points and annex a zero: 46.80.
  2. I add: 128.75 + 46.80 = 175.55 liters.
  3. Now the thrusters use 19.35 liters, so I subtract: 175.55 − 19.35 = 156.20.
  4. The final fuel level is 156.20 liters. I check with estimation: about 130 + 47 − 19 = 158, which is close, so my answer is reasonable.

Practice at home

Match each expression to its answer.

  1. 8.5 + 3.75
  2. 15.2 − 6.8
  3. 4.06 + 7.94

Answer check

  1. 8.5 + 3.75 → 12.25
  2. 15.2 − 6.8 → 8.4
  3. 4.06 + 7.94 → 12

Common mistake

A common mistake in Add and Subtract Decimals is skipping the key idea: "Line up the decimal points, annex zeros if needed, then add or subtract one column at a time." — always check your work against this rule before you submit.

How families can help at home

  • Ask your student to teach you the example above in their own words.
  • Have them say why each step works, not just the answer.
  • Use the sentence starters below so they explain their thinking in full sentences.
