Warm-up

Now let's plot (-5, 2). The x is -5, so which way do we move? Left 5 units.

Teaching notes

Key idea: The signs of (x, y) tell you the quadrant: (+,+) is I, (-,+) is II, (-,-) is III, (+,-) is IV.

I do:

  1. I will plot (-3, -4). I start at the origin (0, 0).
  2. The x is -3, which is negative, so I move 3 units to the LEFT.
  3. The y is -4, which is negative, so I move 4 units DOWN. I place my dot.
  4. Both coordinates are negative, so the point is in Quadrant III.

You do: In the explore, plot each treasure clue like (4, 3) and (-5, 2) and name the quadrant it lands in.

Common misconception

A common mistake in Ordered Pairs in All Four Quadrants is skipping the key idea: "The signs of (x, y) tell you the quadrant: (+,+) is I, (-,+) is II, (-,-) is III, (+,-) is IV." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Quadrant, Negative coordinate, Reflection, Axis, Integer (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

A point has a negative x-coordinate and a negative y-coordinate. Which quadrant is it in?

  • Quadrant I
  • Quadrant II
  • Quadrant III ✓ correct
  • Quadrant IV

Quadrant III is where both x and y are negative. This means the point is left of the origin and below the origin.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension

A city uses a coordinate grid to map locations. City Hall is at the origin (0, 0). The library is at (-3, 4), the park is at (5, -2), the school is at (2, 3), and the hospital is at (-4, -1).
