Unit 4, Lesson 5: Use Percent to Solve Problems6.RP.3c

Everything you need to teach this tomorrow. ~45 min.

Objectives

Content: I can solve problems involving discounts, markups, tax, and tips using percents.

Language: I can explain my solution using the words discount, markup, tax, tip, and percent.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.
  • Preview the key vocabulary below with ESOL students.

Warm-up

A keychain costs 80 tickets with 10% off, then 5% tax.

Teaching notes

Key idea: Take the discount off first to get the new lower price, then add the tax on that new price.

I do:

  1. A bear costs 200 tickets, 25% off, then a 5% tax is added.
  2. First the discount: 0.25 × 200 = 50 off, so the price is 200 − 50 = 150.
  3. Then the tax on the new price: 0.05 × 150 = 7.5.
  4. I add it on: 150 + 7.5 = 157.5 tickets is the final price.

You do: In the table, find the discount, the price after discount, the tax, and the final price. Always take the discount first, then add the tax.

Common misconception

A common mistake in Use Percent to Solve Problems is skipping the key idea: "Take the discount off first to get the new lower price, then add the tax on that new price." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Discount, Markup, Tax, Tip, Percent (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

A jacket costs $80 and is 35% off. Sales tax is 5%. What is the final price?

  • $54.60 ✓ correct
  • $52.00
  • $56.00
  • $49.40

Discount: 0.35 × $80 = $28. Sale price: $80 − $28 = $52. Tax: 0.05 × $52 = $2.60. Final: $52 + $2.60 = $54.60.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension
