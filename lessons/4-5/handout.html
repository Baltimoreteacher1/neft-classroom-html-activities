Use Percent to Solve Problems

6.RP.3c · Unit 4 · Lesson 5

Name   Period

Today's Goal / Meta de hoy

I can solve problems involving discounts, markups, tax, and tips using percents.

Key Idea / Idea clave

Take the discount off first to get the new lower price, then add the tax on that new price.

I Notice / Observo

  • What happens to the price when there is a discount?
  • What happens to the price when tax is added?

I Wonder / Me pregunto

  • Does it matter whether you apply the discount or tax first?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Discount
Descuento		Money taken off the first price to make it cheaper.
Dinero que se quita del precio original para que sea más barato.
Markup
Recargo (margen)		Money added to the cost to set the selling price.
Dinero que se suma al costo para fijar el precio de venta.
Tax
Impuesto		Extra money added to a price for the government.
Dinero extra que se suma a un precio para el gobierno.
Tip
Propina		Extra money you give for good service.
Dinero extra que das por un buen servicio.
Percent
Porcentaje		A way to compare a number to 100, shown with the % sign.
Una manera de comparar un número con 100, con el signo %.

Practice Preview / Vista previa de práctica

  1. Solve each percent word problem.
  2. A store marks up a $30 item by 40%, then a customer gets 10% off. What is the final price?
