# Teacher Guide — Write and Evaluate Expressions

> CardForge package · **DEMO / sample** · status: **draft**

## Overview
Students translate words into algebraic expressions and evaluate expressions by substituting a value for the variable, following the order of operations.

## Objective & Standard
- **Objective:** I can write an expression from words and evaluate it for a given value.
- **Language objective:** I can explain my steps using the words variable, term, and evaluate.
- **Standard:** 6.EE.2
- **Time:** ~45 min

## Materials
- Pencil and practice page

## Pacing
_n/a_

## Warm-Up
If a number is n, write an expression for '4 more than the number.'

## Key Vocabulary (ESOL-friendly)
- **Variable** _(es: Variable)_ — A letter that stands for a number. _(Una letra que representa un número.)_
  - Example: In 3x, x is the variable
- **Coefficient** _(es: Coeficiente)_ — The number multiplied by a variable. _(El número que se multiplica por la variable.)_
  - Example: In 3x, 3 is the coefficient
- **Term** _(es: Término)_ — A number, a variable, or a number times a variable. _(Un número, una variable, o un número por una variable.)_
  - Example: 2x + 5 has two terms: 2x and 5
- **Evaluate** _(es: Evaluar)_ — Find the value by putting in a number for the variable. _(Hallar el valor sustituyendo un número por la variable.)_
  - Example: Evaluate 2x when x = 3 → 6

## Formulas / Definitions
- To evaluate: substitute the value for the variable, then follow the order of operations (powers, then × ÷, then + −).

## Modeled Examples (I do)
**Modeled Example 1.** Evaluate 3x + 5 when x = 4.

   1. Substitute 4 for x: 3(4) + 5.
   2. Multiply first: 3 × 4 = 12.
   3. Then add: 12 + 5 = 17.

   **Answer:** 17

**Modeled Example 2.** Write an expression for 'seven more than a number n.'

   1. 'A number' is n.
   2. 'Seven more than' means add 7.
   3. So the expression is n + 7.

   **Answer:** n + 7

6. Evaluate x squared + 4 (x² + 4) when x = 3.

## Common Misconceptions
- **Adding before multiplying (3x + 5 with x=4 → 3×9).** → Substitute first, then follow order of operations: multiply 3×4 before adding 5.
- **Reading 'x²' as 'x times 2.'** → x² means x times x. So 3² = 3 × 3 = 9, not 6.

## Supports
- **ESOL:** Bilingual EN/ES vocab box; Word-to-symbol chart: 'more than' → +, 'product' → ×, 'twice' → 2×; Sentence frame: 'First I substitute ___, then I ___.'
- **SPED:** Provide the substitution step written out; Allow a calculator for the arithmetic; Reduce to problems 1, 2, 3, 5
