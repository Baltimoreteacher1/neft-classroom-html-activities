Warm-up

Solve x − 4 = 9. Is x being added or subtracted?

Teaching notes

Key idea: Undo the operation with its inverse, and do it to both sides to keep the equation balanced.

I do:

  1. Solve n + 23 = 58. The 23 is added to n.
  2. The inverse of adding 23 is subtracting 23.
  3. I subtract 23 from both sides: n + 23 − 23 = 58 − 23.
  4. That leaves n = 35. Check: 35 + 23 = 58. True!

You do: Use the balance scale to solve n + 23 = 58 by keeping both sides equal. Then solve the practice equations and check each answer.

Common misconception

A common mistake in Solve One-Step Addition and Subtraction Equations is skipping the key idea: "Undo the operation with its inverse, and do it to both sides to keep the equation balanced." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Equation, Inverse operation, Isolate, Solution, One-step equation, Balance (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

Solve: p + 2.8 = 9.1

  • p = 6.3 ✓ correct
  • p = 11.9
  • p = 6.8
  • p = 3.25

Subtract 2.8 from both sides: p = 9.1 − 2.8 = 6.3. Check: 6.3 + 2.8 = 9.1 ✓

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.
