Unit 9, Lesson 3: Compare and Order Integers6.NS.7

Everything you need to teach this tomorrow. ~45 min.

Objectives

Content: I can compare and order integers using a number line.

Language: I can explain my reasoning using the words compare, order, greater than, and less than.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.
  • Preview the key vocabulary below with ESOL students.

Warm-up

Now let's compare -2 and 3. Which one is to the left on the number line? The -2 is left of zero.

Teaching notes

Key idea: The number farther to the left is always less; the number farther to the right is always greater.

I do:

  1. I will compare -7 and -5. They are both negative, so I look at the number line.
  2. -7 is 7 steps left of zero. -5 is only 5 steps left of zero.
  3. -7 is farther to the left, so -7 is the smaller number.
  4. I write -7 < -5. The open side of the symbol points to the bigger number, -5.

You do: In the explore, plot the treasure depths like -2, -7, and -5, then order them from least to greatest.

Common misconception

A common mistake in Compare and Order Integers is skipping the key idea: "The number farther to the left is always less; the number farther to the right is always greater." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Compare, Order, Greater than, Less than, Negative integer, Integer (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

Which list shows the integers in order from LEAST to GREATEST?

  • -2, -5, 0, 3, 7
  • 7, 3, 0, -2, -5
  • -5, -2, 0, 3, 7 ✓ correct
  • 0, -2, -5, 3, 7

-5 is the smallest (farthest left on the number line), then -2, then 0, then 3, then 7. The correct order from least to greatest is -5, -2, 0, 3, 7.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.
