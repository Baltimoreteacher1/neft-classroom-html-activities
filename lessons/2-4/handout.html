Divide Mixed Numbers

6.NS.1 · Unit 2 · Lesson 4

Name   Period

Today's Goal / Meta de hoy

I can divide mixed numbers by first changing them to improper fractions.

Key Idea / Idea clave

Always convert a mixed number to an improper fraction first, then divide using Keep, Change, Flip.

I Notice / Observo

  • What is the total distance of the route?
  • How long is each segment?

I Wonder / Me pregunto

  • Why do we need to convert mixed numbers before dividing?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Mixed Number
Número mixto		A whole number plus a fraction, like 2 1/3.
Un número entero más una fracción, como 2 1/3.
Improper Fraction
Fracción impropia		A fraction where the top is bigger than or equal to the bottom, like 7/4.
Una fracción donde el número de arriba es mayor o igual al de abajo, como 7/4.
Convert
Convertir		To change a number to a new form but keep the same value.
Cambiar un número a otra forma sin cambiar su valor.
Simplify
Simplificar		To make a fraction smaller using the same parts, like 2/4 = 1/2.
Hacer una fracción más pequeña con las mismas partes, como 2/4 = 1/2.
Reciprocal
Recíproco		A fraction turned upside down. You use it in keep-change-flip.
Una fracción volteada de arriba abajo. Se usa en deja-cambia-voltea.
