Unit 2, Lesson 4: Divide Mixed Numbers6.NS.1

Objectives

Content: I can divide mixed numbers by first changing them to improper fractions.

Language: I can explain my steps using the words mixed number, improper fraction, convert, reciprocal, and simplify.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.

Warm-up

Let's try 2 1/2 ÷ 1/2. First convert 2 1/2: 2 × 2 = 4, plus 1 = 5, so 2 1/2 = 5/2.

Teaching notes

Key idea: Always convert a mixed number to an improper fraction first, then divide using Keep, Change, Flip.

I do:

  1. Agent Chen has a 3 1/2-mile route cut into 1/4-mile segments. I want 3 1/2 ÷ 1/4.
  2. First I convert 3 1/2. Multiply the whole by the bottom: 3 × 2 = 6. Add the top: 6 + 1 = 7. So 3 1/2 = 7/2.
  3. Now Keep, Change, Flip: keep 7/2, change ÷ to ×, flip 1/4 to 4/1.
  4. Multiply: 7/2 × 4/1 = 28/2.
  5. Simplify: 28/2 = 14. So 3 1/2 ÷ 1/4 = 14 segments.

You do: Now fill in the table: convert each mixed number to an improper fraction, then divide using Keep, Change, Flip.

Common misconception

A common mistake in Divide Mixed Numbers is skipping the key idea: "Always convert a mixed number to an improper fraction first, then divide using Keep, Change, Flip." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Mixed Number, Improper Fraction, Convert, Simplify, Reciprocal (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

What is 2 2/3 ÷ 2/3?

2 2/3 = 8/3. Then 8/3 ÷ 2/3 = 8/3 × 3/2 = 24/6 = 4.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.
