Unit 8, Lesson 4: Appropriate Measures6.SP.5d

Everything you need to teach this tomorrow. ~45 min.

Objectives

Content: I can choose the best measure of center for a data set based on its shape.

Language: I can explain my choice using the words mean, median, outlier, and skewed.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.
  • Preview the key vocabulary below with ESOL students.

Warm-up

Data set A: 4, 5, 6, 7, 8. Are there any outliers? If not, which measure should we use, mean or median?

Teaching notes

Key idea: Use the mean for data with no outliers; use the median when an outlier or a skewed shape would pull the mean away from typical.

I do:

  1. My data, the top scorer's points: 22, 24, 20, 25, 23, 21, 58. I notice 58 is much bigger than the rest, so it is an outlier.
  2. Mean: 22 + 24 + 20 + 25 + 23 + 21 + 58 = 193, then 193 ÷ 7 ≈ 27.6.
  3. Median: in order they are 20, 21, 22, 23, 24, 25, 58. The middle (4th) value is 23.
  4. The mean (27.6) is higher than 6 of the 7 games because 58 pulled it up. So the median (23) better shows a typical game.

You do: Next, you will sort sports scenarios into "Use Mean" or "Use Median." For each one, look for an outlier or a skewed shape to decide.

Common misconception

A common mistake in Appropriate Measures is skipping the key idea: "Use the mean for data with no outliers; use the median when an outlier or a skewed shape would pull the mean away from typical." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Mean, Median, Outlier, Skewed, Data distribution, Variability (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

Data set: 15, 18, 16, 17, 15, 72. Which measure of center best represents the data?

  • Median, because 72 is an outlier ✓ correct
  • Mean, because it uses all values
  • Mean, because it is always best
  • Neither works for this data

The value 72 is an outlier. The mean (25.5) is pulled high by 72 and doesn't represent the typical values. The median (16.5) is a better measure of center.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension

A sports reporter writes: 'The average ticket price for the playoffs is $85.' The actual prices are: $45, $50, $48, $52, $55, $260. The $260 ticket is a courtside seat.
