Unit 8, Lesson 3: Mean Absolute Deviation6.SP.5c

A quick guide for families and for any student who missed class.

What students are learning

The mean absolute deviation, or MAD, is the average distance of each value from the mean. It tells you how spread out the data is: a small MAD means the data is close together, and a large MAD means it is spread out.

Key idea: MAD = the average of how far each number is from the mean (always a positive distance).

Key words

  • Mean Absolute Deviation — The average distance of each number from the mean.
    Español: Desviación media absoluta — La distancia promedio de cada número a la media.
  • Deviation — How far a number is from the mean.
    Español: Desviación — Qué tan lejos está un número de la media.
  • Absolute Value — How far a number is from zero. It is always positive.
    Español: Valor absoluto — Qué tan lejos está un número de cero. Siempre es positivo.
  • Spread — How far apart the numbers are.
    Español: Dispersión — Qué tan separados están los números.
  • Data distribution — How the data looks: where it sits and how spread out it is.
    Español: Distribución de datos — Cómo se ven los datos: dónde están y qué tan separados están.
  • Variability — How spread out the numbers are.
    Español: Variabilidad — Qué tan separados están los números.

Example

  1. My data set: 6, 8, 10, 12. First I find the mean: 6 + 8 + 10 + 12 = 36, then 36 ÷ 4 = 9. The mean is 9.
  2. Next I find each deviation (value − mean): 6−9 = −3, 8−9 = −1, 10−9 = 1, 12−9 = 3.
  3. Then I take the absolute value of each (the distance, always positive): 3, 1, 1, 3.
  4. Finally I average those distances: 3 + 1 + 1 + 3 = 8, then 8 ÷ 4 = 2. So the MAD = 2 points.

Practice at home

Match each MAD value to what it tells you about the data.

  1. MAD = 1.5
  2. MAD = 8.0
  3. MAD = 0

Answer check

  1. MAD = 1.5 → Very consistent data
  2. MAD = 8.0 → Very spread out data
  3. MAD = 0 → All values are identical
