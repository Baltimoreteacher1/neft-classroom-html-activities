Unit 8, Lesson 3: Mean Absolute Deviation6.SP.5c

Objectives

Content: I can find the mean absolute deviation (MAD) to describe how spread out data is.

Language: I can explain my work using the words mean absolute deviation, deviation, absolute value, and spread.

Warm-up

New data set: 9, 10, 11. First, what is the mean? Add the three numbers and divide by 3.

Teaching notes

Key idea: MAD = the average of how far each number is from the mean (always a positive distance).

I do:

  1. My data set: 6, 8, 10, 12. First I find the mean: 6 + 8 + 10 + 12 = 36, then 36 ÷ 4 = 9. The mean is 9.
  2. Next I find each deviation (value − mean): 6−9 = −3, 8−9 = −1, 10−9 = 1, 12−9 = 3.
  3. Then I take the absolute value of each (the distance, always positive): 3, 1, 1, 3.
  4. Finally I average those distances: 3 + 1 + 1 + 3 = 8, then 8 ÷ 4 = 2. So the MAD = 2 points.

You do: Next, you will find the MAD for Player A's scores step by step in the table. Remember: deviation first, then absolute value, then average.

Common misconception

A common mistake in Mean Absolute Deviation is skipping the key idea: "MAD = the average of how far each number is from the mean (always a positive distance)." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Mean Absolute Deviation, Deviation, Absolute Value, Spread, Data distribution, Variability (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”

Exit ticket

Data set: 4, 8, 6, 10, 2. The mean is 6. What is the MAD?

  • 2.4 ✓ correct
  • 6
  • 0
  • 12

Deviations: −2, 2, 0, 4, −4. Absolute deviations: 2, 2, 0, 4, 4. Sum = 12. MAD = 12 ÷ 5 = 2.4.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension

Two soccer goalkeepers track goals allowed per game over 6 games. Keeper A: 1, 2, 1, 3, 2, 1 (mean = 1.67, MAD ≈ 0.56). Keeper B: 0, 4, 1, 3, 0, 2 (mean = 1.67, MAD ≈ 1.22). The coach needs to pick a starter for the championship game.
