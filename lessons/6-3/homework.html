Write Algebraic Expressions

Target I can write algebraic expressions from words and real-world situations.
Discuss I can explain my expression using the words variable, coefficient, constant, and algebraic expression.

🔑 Key Vocabulary

Variable

Variable

💡 Example: In 5t + 20, t could be hours worked — if t = 3, the expression equals 35; if t = 8, it equals 60
Tap to show definition ➔

A letter that stands for a number that is unknown or can change.

Una letra que representa un número desconocido o que puede cambiar.

Algebraic Expression

Expresión algebraica

💡 Example: 3n + 7 means 'triple a number, then add 7' — it has a variable (n), a coefficient (3), and a constant (7)
Tap to show definition ➔

A math phrase that has at least one letter.

Una frase matemática que tiene al menos una letra.

Coefficient

Coeficiente

💡 Example: In 8x, the coefficient 8 means '8 groups of x' — if x = 3, then 8x = 8 × 3 = 24
Tap to show definition ➔

The number in front of a letter, like the 3 in 3x.

El número frente a una letra, como el 3 en 3x.

Constant

Constante

💡 Example: In 3n + 7, the 7 never changes no matter what n is — like a flat fee that stays the same
Tap to show definition ➔

A number on its own that does not change.

Un número solo que no cambia.

Like terms

Términos semejantes

💡 Example: 4x + 2x = 6x (like terms, same variable); but 4x + 2y cannot be combined (different variables)
Tap to show definition ➔

Terms with the same letter, like 2x and 5x.

Términos con la misma letra, como 2x y 5x.

Problem 10
ERROR ANALYSIS

⚠️ Find the Translation Error

Review the steps below. Identify which step contains the error, and explain why.

Step 1 Phrase: 12 less than a number g
Step 2 Student's expression: 12 − g
Step 3 Reasoning: I see 'less than' so I subtract, and 12 comes first in the phrase.
Correct Work Reference: '12 less than a number g' means start with g and subtract 12: g − 12. The phrase 'less than' reverses the order — the number being subtracted FROM comes after 'less than.'
