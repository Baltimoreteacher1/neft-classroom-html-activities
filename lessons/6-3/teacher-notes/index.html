Warm-up

Teaching notes

Key idea: Words like "each" or "per" mean multiply, and "plus" or "more than" mean add.

I do:

  1. Tickets cost $12 each, plus a flat $50 venue fee. Let t be the number of tickets.
  2. "$12 each ticket" means 12 times t, which I write as 12t.
  3. The flat fee $50 is added once, so I add 50.
  4. I put them together: total revenue = 12t + 50.

You do: Sort each word phrase into the matching expression in the activity. Remember: "more than" adds and "product" multiplies.

Common misconception

A common mistake in Write Algebraic Expressions is skipping the key idea: "Words like "each" or "per" mean multiply, and "plus" or "more than" mean add." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Variable, Algebraic Expression, Coefficient, Constant, Like terms (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

Which expression represents '3 more than twice a number n'?

  • 2n + 3 ✓ correct
  • 3n + 2
  • 2(n + 3)
  • 2 + 3n

'Twice a number n' is 2n. '3 more than' means + 3. So the expression is 2n + 3.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.
