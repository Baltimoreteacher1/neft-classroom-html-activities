Powers and Exponents

6.EE.1 · Unit 6 · Lesson 1

Name   Period

Today's Goal / Meta de hoy

I can write and evaluate numbers in exponent form using a base and a power.

Key Idea / Idea clave

A power like 2³ is a short way to write repeated multiplication: 2 × 2 × 2.

I Notice / Observo

  • What happens to the volume each time you flip a switch?
  • Why is 2³ equal to 8, not 6?

I Wonder / Me pregunto

  • What if the volume tripled each time instead of doubled?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Exponent
Exponente		A small number that tells how many times to multiply the number by itself.
Un número pequeño que dice cuántas veces multiplicar el número por sí mismo.
Base
Base		The number that gets multiplied by itself.
El número que se multiplica por sí mismo.
Power
Potencia		A number written with a base and an exponent, like 2³.
Un número escrito con una base y un exponente, como 2³.
Evaluate
Evaluar		To find the value of an expression.
Encontrar el valor de una expresión.
Coefficient
Coeficiente		The number in front of a letter, like the 3 in 3x.
El número frente a una letra, como el 3 en 3x.
Like terms
Términos semejantes		Terms with the same letter, like 2x and 5x.
Términos con la misma letra, como 2x y 5x.
