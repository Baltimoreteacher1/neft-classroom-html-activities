Unit 7, Lesson 6: Solve and Graph Inequalities6.EE.5

A quick guide for families and for any student who missed class.

What students are learning

I solve an inequality the same way as an equation: use inverse operations to get the variable alone. But the answer is a range of values, so I show all of them on a number line.

Key idea: Solve with inverse operations, then graph the solution set: open circle for < or >, closed for ≤ or ≥.

Key words

  • Solve — To find the number that makes it true.
    Español: Resolver — Encontrar el número que la hace verdadera.
  • Graph — To show answers on a number line with circles and shading.
    Español: Graficar — Mostrar respuestas en una recta numérica con círculos y sombreado.
  • Solution — A number that makes the equation or inequality true.
    Español: Solución — Un número que hace verdadera la ecuación o desigualdad.
  • Substitute — To put a number in for the letter to check if it works.
    Español: Sustituir — Poner un número en lugar de la letra para ver si funciona.
  • Solution set — All of the values that make an inequality true.
    Español: Conjunto solución — Todos los valores que hacen verdadera una desigualdad.
  • Inverse operation — An operation that undoes another, used to isolate the variable.
    Español: Operación inversa — Una operación que deshace otra, usada para despejar la variable.

Example

  1. Solve x + 3 > 53. The inverse of adding 3 is subtracting 3.
  2. I subtract 3 from both sides: x + 3 − 3 > 53 − 3, so x > 50.
  3. The symbol > does not include 50, so I use an open circle at 50.
  4. 'Greater than' shades right. Check: x = 60 gives 60 + 3 = 63 > 53. True!

Practice at home

  1. Solve and describe the graph: x + 5 < 11

Answer check

  1. Solve and describe the graph: x + 5 < 11 → x < 6; open circle at 6, shade left

Common mistake

A common mistake in Solve and Graph Inequalities is skipping the key idea: "Solve with inverse operations, then graph the solution set: open circle for < or >, closed for ≤ or ≥." — always check your work against this rule before you submit.

How families can help at home

  • Ask your student to teach you the example above in their own words.
  • Have them say why each step works, not just the answer.
  • Use the sentence starters below so they explain their thinking in full sentences.
