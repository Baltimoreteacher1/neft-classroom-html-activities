Ratio Tables

6.RP.3a · Unit 3 · Lesson 2

Name   Period

Today's Goal / Meta de hoy

I can use a ratio table to find equivalent ratios.

Key Idea / Idea clave

To keep a ratio the same, multiply both numbers by the same scale factor.

I Notice / Observo

  • What two quantities are being compared in the recipe?
  • What happens to both quantities when you double the recipe?

I Wonder / Me pregunto

  • What if we only needed to make half the original recipe?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Ratio table
Tabla de razones		A table of equal ratios that shows a pattern.
Una tabla de razones iguales que muestra un patrón.
Equivalent ratio
Razón equivalente		Two ratios that mean the same thing, like 1:2 and 2:4.
Dos razones que significan lo mismo, como 1:2 y 2:4.
Scale factor
Factor de escala		The number you multiply both parts of a ratio by to get an equal ratio.
El número por el que multiplicas ambas partes de una razón para obtener una razón igual.
Pattern
Patrón		A rule that numbers follow again and again.
Una regla que los números siguen una y otra vez.
Additive pattern
Patrón aditivo		Adding the same amount to both parts of a ratio table to make new rows.
Sumar la misma cantidad a ambas partes de una tabla de razones para hacer filas nuevas.
