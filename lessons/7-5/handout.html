Graph Inequalities

6.EE.8 · Unit 7 · Lesson 5

Name   Period

Today's Goal / Meta de hoy

I can graph the solutions of an inequality on a number line.

Key Idea / Idea clave

Closed circle for ≤ or ≥ (included), open circle for < or > (not included); shade right for greater, left for less.

I Notice / Observo

  • What does 'no earlier than 2:00 PM' mean about the time?
  • Should 2:00 PM itself be included?

I Wonder / Me pregunto

  • What would the graph look like if the suspect arrived before 2:00 PM?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Number line
Recta numérica		A line with numbers in order. You use it to show answers.
Una línea con números en orden. La usas para mostrar respuestas.
Open circle
Círculo abierto		An empty circle showing a number is NOT included.
Un círculo vacío que muestra que un número NO está incluido.
Closed circle
Círculo cerrado		A filled-in circle showing a number IS included.
Un círculo relleno que muestra que un número SÍ está incluido.
Solution set
Conjunto solución		All the numbers that make the inequality true.
Todos los números que hacen verdadera la desigualdad.
Inequality
Desigualdad		A math sentence comparing two amounts with <, >, ≤, or ≥.
Una oración matemática que compara dos cantidades con <, >, ≤ o ≥.
Boundary point
Punto límite		The number on the number line where the solution set starts.
El número en la recta numérica donde comienza el conjunto solución.

Practice Preview / Vista previa de práctica

  1. Match each inequality to its graph description
  2. Read the graph description and write the inequality it represents. Then graph it.
