I can…

I can graph the solutions of an inequality on a number line.

Key words

  • Number line — A line with numbers in order. You use it to show answers.
  • Open circle — An empty circle showing a number is NOT included.
  • Closed circle — A filled-in circle showing a number IS included.
  • Solution set — All the numbers that make the inequality true.
  • Inequality — A math sentence comparing two amounts with <, >, ≤, or ≥.
  • Boundary point — The number on the number line where the solution set starts.

Steps

  1. Graph t ≥ 2. The boundary number is 2.
  2. The symbol ≥ includes 2, so I draw a closed (filled) circle at 2.
  3. 'Greater than or equal to' means larger values work, so I shade to the right.
  4. Now the graph shows 2, 3, 4, and every value above 2.
