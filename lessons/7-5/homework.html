Graph Inequalities

Target I can graph the solutions of an inequality on a number line.
Discuss I can explain my graph using the words number line, open circle, closed circle, and solution set.

🔑 Key Vocabulary

Number line

Recta numérica

💡 Example: A horizontal line with marks at 0, 1, 2, 3, ... showing where values fall
Tap to show definition ➔

A line with numbers in order. You use it to show answers.

Una línea con números en orden. La usas para mostrar respuestas.

Open circle

Círculo abierto

💡 Example: x > 5: open circle at 5 means 5 itself is NOT a solution, but 5.1, 6, 7, ... are
Tap to show definition ➔

An empty circle showing a number is NOT included.

Un círculo vacío que muestra que un número NO está incluido.

Closed circle

Círculo cerrado

💡 Example: x ≥ 5: closed circle at 5 means 5 IS a solution, and so are 6, 7, 8, ...
Tap to show definition ➔

A filled-in circle showing a number IS included.

Un círculo relleno que muestra que un número SÍ está incluido.

Solution set

Conjunto solución

💡 Example: For x > 3, the solution set includes 3.1, 4, 5, 10, 100 — any number greater than 3
Tap to show definition ➔

All the numbers that make the inequality true.

Todos los números que hacen verdadera la desigualdad.

Inequality

Desigualdad

💡 Example: x ≥ 3 is an inequality; it is graphed as a closed circle on 3 with an arrow to the right.
Tap to show definition ➔

A math sentence comparing two amounts with <, >, ≤, or ≥.

Una oración matemática que compara dos cantidades con <, >, ≤ o ≥.

Boundary point

Punto límite

💡 Example: For x > 5, the boundary point is 5, shown with an open circle.
Tap to show definition ➔

The number on the number line where the solution set starts.

El número en la recta numérica donde comienza el conjunto solución.

⚠️ Find the Graphing Error

Review the steps below. Identify which step contains the error, and explain why.

Step 1 Inequality: x > 6
Step 2 Circle type: Closed circle at 6
Step 3 Shade direction: Shade right
Step 4 Graph: Closed circle at 6, shaded right
Correct Work Reference: x > 6 uses a strict inequality (no 'equal to'), so the circle should be OPEN, not closed. Open circle at 6, shaded right.
