Unit 8, Lesson 1: Statistical Questions and Data6.SP.1

Everything you need to teach this tomorrow. ~45 min.

Objectives

Content: I can tell the difference between a statistical question and a non-statistical question.

Language: I can explain my reasoning using the words statistical question, data, variability, and survey.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.
  • Preview the key vocabulary below with ESOL students.

Warm-up

New question: "How many pets does each student in our class have?" Will every student give the same answer, or different answers?

Teaching notes

Key idea: If you expect the answers to vary, it is a statistical question.

I do:

  1. Question 1: "How many points did our top player score last night?" There is one player and one game, so there is one fixed answer, like 18 points.
  2. One answer, no variety, so this is NOT a statistical question.
  3. Question 2: "How many points does each player score per game?" The answers vary: 8, 12, 12, 15, 18, 20, 22.
  4. Because the answers are different for different players, I must collect data. This IS a statistical question.

You do: Next, you will sort questions into Statistical or Not Statistical, just like I did. For each one, ask yourself: will the answers be different, or always the same?

Common misconception

A common mistake in Statistical Questions and Data is skipping the key idea: "If you expect the answers to vary, it is a statistical question." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Statistical Question, Data, Variability, Survey, Data distribution (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

Which question is a statistical question? A) How many students are in the school? B) How many hours does each student in 6th grade exercise per week? C) What time does school start? D) How many days are in a week?

  • B) How many hours does each student exercise per week? ✓ correct
  • A) How many students are in the school?
  • C) What time does school start?
  • D) How many days are in a week?

"How many hours does each student exercise per week?" is statistical because different students exercise different amounts — the data has variability.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension

The school newspaper wants to write a story about the girls' soccer team. A reporter suggests asking: "How many goals has the team scored total?" The editor says they should ask: "How many goals did each player score this season?"
