I can use the distributive property to expand and factor expressions.

Key words

  • Distributive Property — Multiplying a number by everything inside the parentheses: a(b + c) = ab + ac.
  • Factor — A number that gets multiplied by another number.
  • Expand — To multiply out the parentheses in an expression.
  • Equivalent — Expressions that always have the same value.
  • Coefficient — The number in front of a letter, like the 3 in 3x.
  • Like terms — Terms with the same letter, like 2x and 5x.

Steps

  1. Each bundle has a ticket ($15) and a snack ($5). For 3 bundles I can write 3(15 + 5).
  2. Way 1: add inside first. 15 + 5 = 20, then 3 × 20 = 60.
  3. Way 2: distribute the 3 to each part. 3 × 15 + 3 × 5.
  4. That is 45 + 15 = 60.
  5. Both ways give $60, so 3(15 + 5) = 3 × 15 + 3 × 5.
