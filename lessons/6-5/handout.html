The Distributive Property

6.EE.3 · Unit 6 · Lesson 5

Name   Period

Today's Goal / Meta de hoy

I can use the distributive property to expand and factor expressions.

Key Idea / Idea clave

The number outside the parentheses gets "shared" with every term inside.

I Notice / Observo

  • Why do both methods give the same total?
  • What does the 3 represent in 3(15 + 5)?

I Wonder / Me pregunto

  • What if 10 people bought bundles — which way would be faster to calculate?

Vocabulary / Vocabulario

Term / TérminoDefinition / Definición
Distributive Property
Propiedad distributiva		Multiplying a number by everything inside the parentheses: a(b + c) = ab + ac.
Multiplicar un número por todo lo que está dentro del paréntesis: a(b + c) = ab + ac.
Factor
Factor		A number that gets multiplied by another number.
Un número que se multiplica por otro número.
Expand
Desarrollar		To multiply out the parentheses in an expression.
Multiplicar lo que está dentro del paréntesis en una expresión.
Equivalent
Equivalente		Expressions that always have the same value.
Expresiones que siempre tienen el mismo valor.
Coefficient
Coeficiente		The number in front of a letter, like the 3 in 3x.
El número frente a una letra, como el 3 en 3x.
Like terms
Términos semejantes		Terms with the same letter, like 2x and 5x.
Términos con la misma letra, como 2x y 5x.

Practice Preview / Vista previa de práctica

  1. Expand 6(n + 3) using the distributive property.
    1. 6n + 18
    2. 6n + 3
    3. n + 18
    4. 6n + 9
  2. Use the distributive property to expand each expression.
  3. Match each factored form to its expanded form.
  4. Find the Distributive Error
