I can…

I can find the distance between two points on the coordinate plane using absolute value.

Key words

  • Distance — How far apart two points are. It is never negative.
  • Absolute value — How far a number is from zero.
  • Horizontal distance — How far apart two points are going left or right.
  • Vertical distance — How far apart two points are going up or down.
  • Coordinate plane — A grid with a line going across and a line going up that cross.
  • Integer — Whole numbers and their opposites, like -2, -1, 0, 1, 2.

Steps

  1. I will find the distance from (-3, 2) to (4, 2). Both have y = 2, so they sit on the same horizontal line.
  2. The points are on opposite sides of zero, so I add the absolute values.
  3. |-3| = 3 and |4| = 4.
  4. I add: 3 + 4 = 7. The distance between them is 7 units.

Try it

What is the distance between the points (3, -2) and (3, 5)?

Check your answer

7 units

Both points share x = 3, so this is a vertical distance. |5 - (-2)| = |5 + 2| = 7 units.
