Unit 8, Lesson 6: Display Data: Histograms6.SP.4

A quick guide for families and for any student who missed class.

What students are learning

A histogram is a bar graph that groups numbers into equal ranges called intervals. The height of each bar is the frequency: how many values fall in that interval. The bars touch because the intervals are continuous.

Key idea: Each bar's height shows how many data values land inside that interval.

Key words

  • Histogram — A bar graph that groups data into equal ranges. The bars touch.
    Español: Histograma — Una gráfica de barras que agrupa datos en rangos iguales. Las barras se tocan.
  • Frequency — How many times a value shows up.
    Español: Frecuencia — Cuántas veces aparece un valor.
  • Interval — A range of numbers used to group data.
    Español: Intervalo — Un rango de números para agrupar datos.
  • Distribution — How the data is spread out.
    Español: Distribución — Cómo están repartidos los datos.
  • Data distribution — How the data looks: where it sits and how spread out it is.
    Español: Distribución de datos — Cómo se ven los datos: dónde están y qué tan separados están.
  • Variability — How spread out the numbers are.
    Español: Variabilidad — Qué tan separados están los números.

Example

  1. My data: 3, 7, 12, 15, 18, 24. I will use intervals of 10: 0–9, 10–19, 20–29.
  2. Interval 0–9: the values 3 and 7 fit, so the frequency is 2.
  3. Interval 10–19: the values 12, 15, and 18 fit, so the frequency is 3.
  4. Interval 20–29: only the value 24 fits, so the frequency is 1.
  5. Now I draw three bars with heights 2, 3, and 1, touching each other. The tallest bar (10–19) is where most values are.

Answer check

  1. Histogram → Bars touch, shows frequency by interval
  2. Bar graph → Bars separated, shows categories
  3. Frequency → Count of values in an interval
