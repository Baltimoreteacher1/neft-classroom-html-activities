Learn It

Bars over number ranges

A histogram groups data into equal ranges called bins (like 0–9, 10–19). Each bar's height tells how many values landed in that range. Read the height, and add the heights for the total.

Which bin? the value 14 belongs in the 10–19 bin, because 14 is between 10 and 19.
Total values: if the bars are 3, 5, 2, add them: 3 + 5 + 2 = 10 values in all.

Hint

Level 1 Sort numbers into bins, then count.

A. Which bin does the value 8 go in?

Hint

8 is between 0 and 9.

B. Three bars count 4, 2, and 3. What is the total number of values?

Hint

4 + 2 = 6, then 6 + 3 = 9.

Level 2 Build counts from data.

A. These values are 0–9: 3, 5, 8, 1. How many values are in the 0–9 bin?

Hint

Count them: 3, 5, 8, 1 → that's 4 values.

B. Histogram bars: 2, 4, 6, 3. What is the total number of values?

Hint

2 + 4 = 6, + 6 = 12, + 3 = 15.
