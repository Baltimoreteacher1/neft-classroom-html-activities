Unit 8, Lesson 6: Display Data: Histograms6.SP.4

Everything you need to teach this tomorrow. ~45 min.

Objectives

Content: I can make and read a histogram to display data in intervals.

Language: I can explain my histogram using the words histogram, frequency, interval, and distribution.

Before class

  • Print / open: guided notes, homework, slides.
  • Have the family page link ready for absent students.
  • Preview the key vocabulary below with ESOL students.

Warm-up

New data: 2, 4, 8, 11, 13, 16, 21. We will use intervals of 10 again: 0–9, 10–19, 20–29.

Teaching notes

Key idea: Each bar's height shows how many data values land inside that interval.

I do:

  1. My data: 3, 7, 12, 15, 18, 24. I will use intervals of 10: 0–9, 10–19, 20–29.
  2. Interval 0–9: the values 3 and 7 fit, so the frequency is 2.
  3. Interval 10–19: the values 12, 15, and 18 fit, so the frequency is 3.
  4. Interval 20–29: only the value 24 fits, so the frequency is 1.
  5. Now I draw three bars with heights 2, 3, and 1, touching each other. The tallest bar (10–19) is where most values are.

You do: Next, you will build a frequency table for all 30 players' scoring averages using intervals of 10. Then you will read the histogram shape to see where most players cluster.

Common misconception

A common mistake in Display Data: Histograms is skipping the key idea: "Each bar's height shows how many data values land inside that interval." — always check your work against this rule before you submit.

ESOL / SPED supports

  • Pre-teach: Histogram, Frequency, Interval, Distribution, Data distribution, Variability (Spanish + visuals are in the lesson vocabulary).
  • Offer the sentence frames: “I know ___ because ___.” “First, I ___. Then, I ___.” “The answer is ___, so ___.”
  • Chunk the I-do steps; let students restate each step to a partner before moving on.
  • Provide the worked example as a reference while they practice.

Exit ticket

A histogram of player heights shows: 60–63 in: 2, 64–67 in: 7, 68–71 in: 9, 72–75 in: 4. Which interval has the most players?

  • 68–71 inches ✓ correct
  • 64–67 inches
  • 72–75 inches
  • 60–63 inches

The 68–71 inch interval has the highest frequency of 9 players.

Reteach

Re-run the I-do example with smaller numbers and have students narrate each step using the key idea.

Extension

A soccer coach looks at a histogram of shots-per-game for the team: 0–2 shots: 1 player, 3–5 shots: 8 players, 6–8 shots: 10 players, 9–11 shots: 3 players, 12–14 shots: 1 player. The coach says, 'Most of my players take a good number of shots.'
