Decimal Docking Bay: Deep Space

Command rank, Cadet. These missions chain several decimal operations and ask you to reason about place value, not just compute. Think it through and launch the fleet.

Rango de mando, Cadete. Estas misiones encadenan operaciones con decimales y piden razonar sobre el valor posicional. Piensa bien y lanza la flota.

Act 1 · Lesson 1-5

The Supply Manifest

📦 Manifest Lock #1 — Add Three Decimals

Add all three crates: 2.45 + 1.8 + 0.75. What is the total mass?Suma las tres cajas: 2.45 + 1.8 + 0.75. ¿Cuál es la masa total?

💡 Hint: Annex a zero so 1.8 = 1.80. Then 2.45 + 1.80 + 0.75. Add the hundredths first.
Sentence frame: “Lining up all three, 2.45 + 1.80 + 0.75 = ____ kg.”

📦 Manifest Lock #2 — Remaining Capacity

The cargo limit is 6 kg. With 5.00 kg loaded, how much more mass can you add?El límite es 6 kg. Con 5.00 kg cargados, ¿cuánta masa más cabe?

💡 Hint: Subtract: 6.00 - 5.00. Annex zeros so both have two decimal places.
Sentence frame: “6.00 - 5.00 = ____ kg of room left.”

Act 2 · Lesson 1-6

Thruster Calibration

🔥 Thruster Lock #1 — Multiply Decimals

Compute 1.2 × 0.25 to find the plasma used. What is the product?Calcula 1.2 × 0.25 para el plasma usado. ¿Cuál es el producto?

💡 Hint: 12 × 25 = 300. Decimal places: 1 (in 1.2) + 2 (in 0.25) = 3, so 0.300 = 0.3.
Sentence frame: “12 × 25 = 300, and with 3 decimal places that is 0.3 units.”

🔥 Thruster Lock #2 — Scale the Array

The array has 4 thrusters. If one uses 0.3 units, the array uses 4 × 0.3. How much plasma total?El arreglo tiene 4 propulsores. Si uno usa 0.3, el arreglo usa 4 × 0.3. ¿Cuánto total?

💡 Hint: 4 × 3 = 12, and 0.3 has 1 decimal place, so the product has 1 decimal place: 1.2.
Sentence frame: “4 × 0.3 = ____ units of plasma for the whole array.”
🔒 Finish the previous chapter first to unlock this one.
Final · Lessons 1-5 & 1-6

The Fleet Launch Code

🔒 Fleet Code — Multi-Step

Compute (3.25 + 0.75) × 0.5. Which line is fully correct?Calcula (3.25 + 0.75) × 0.5. ¿Cuál línea es correcta?

💡 Hint: 3.25 + 0.75 = 4.00. Then 4.00 × 0.5 = 2.0 (half of 4).
Sentence frame: “3.25 + 0.75 = ____, and that sum × 0.5 = ____.”
