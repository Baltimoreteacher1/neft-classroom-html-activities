Detective Case Files: The Equation Mysteries

Detective Mateo has inherited the precinct's coldest, trickiest cases. Every locked evidence box guards an unknown quantity. Your job: translate layered clues into equations, solve them, and verify each answer before the lock will yield. Only airtight reasoning cracks these files.

⭐ Each act hides an optional Detective’s Challenge. It is bonus glory only — you can skip it anytime and still solve the case.

Act 1 · Write Equations

The First Clue

🕵️ Clue #1 — Model the Theft

The case file reads: “The vault held some bills. The thief took half of them, then a guard recovered 8, leaving 30 in the vault.” Let b = the original number of bills. Which equation models the clue, in order?

🕵️ Clue #2 — The Fingerprint Count

Lab note: “Three more than four times the number of usable fingerprints (f) equals 23.” Translate it into an equation.

⭐ Detective’s Challenge

The Double-Cross

Optional — pure bonus. The case is already cracked; skip anytime with the button below.

A coded confession says: “Twice a number, decreased by 7, gives the same result as the number increased by 5.” Which equation captures it?

Act 2 · Solve Equations

The Locked Boxes

🔐 Box #1 — Solve to Unlock

The first lock opens to the value of x. Isolate the variable and find the code.

x / 6 = 9

🔐 Box #2 — The Stubborn Lock

This lock needs two moves. Solve for n, then verify by substituting back.

3n + 5 = 26
⭐ Detective’s Challenge

The Verification Trap

Optional — pure bonus. Both boxes are already open; skip anytime with the button below.

A tampered file claims a solution. Use substitution to find the value of m that actually makes the equation true.

8 = m − 15
🔒 Complete Act 2 to unlock the Final Case.
Final Case · Boss

The Vault Mystery

🔒 The Vault — Write, Solve, Verify

The master clue: “When you take twice the number of stolen coins and subtract 12, you are left with 30.” Let c = stolen coins. Pick the line whose equation AND solution are both correct.

⭐ Detective’s Final Challenge

The Inner Safe

Optional — pure bonus. The vault is already open and the case solved; skip anytime with the button below.

Behind the vault sits a smaller safe. Its code y satisfies the clue below. Solve and verify.

y / 4 + 6 = 13
