Notes + Student Practice

Median is the middle value after numbers are sorted from least to greatest. Mode is the value that appears most often.

Use (N + 1) ÷ 2 to find the median position. These notes give the steps, but you do the calculating.

Median Steps

  1. Write the numbers in order.
  2. Count the values. This is N.
  3. Use (N + 1) ÷ 2.
  4. If the answer is a whole number, use that position.
  5. If the answer ends in .5, average the two middle values.

Mode Steps

  1. Look for repeated values.
  2. Count how many times each value appears.
  3. The value that appears most is the mode.
  4. If nothing repeats, write no mode.

Helpful Frames

Guided Calculation

  1. Click Combine + Calculate to put all 6 goal totals together.
  2. Write the 6 numbers in order from least to greatest.
  3. Count the numbers: N = 6.
  4. Use (6 + 1) ÷ 2 = 3.5, so the median is between the 3rd and 4th numbers.
  5. Average the 3rd and 4th numbers to get the median.
  6. Look for a number that repeats to find the mode.
