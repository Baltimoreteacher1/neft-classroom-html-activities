Simple Notes: Median and Mode

Median is the middle value after the numbers are sorted from least to greatest. Use (N + 1) ÷ 2 to find the median position.

Mode is the value that appears the most. There can be one mode, more than one mode, or no mode.

Odd number of values

  1. Sort the values.
  2. Count the values. That count is N.
  3. Use (N + 1) ÷ 2.
  4. If the answer is a whole number, use that position.

Example: 1, 2, 2, 2, 3

N = 5, so (5 + 1) ÷ 2 = 3. The median is the 3rd value: 2.

Even number of values

  1. Sort the values.
  2. Count the values. That count is N.
  3. Use (N + 1) ÷ 2.
  4. If the answer ends in .5, average the two middle values around it.

Example: 1, 2, 2, 3

N = 4, so (4 + 1) ÷ 2 = 2.5. Average the 2nd and 3rd values: (2 + 2) ÷ 2 = 2.

Sentence frames

Median: The median position is ___ because (N + 1) ÷ 2 = ___.

Even data: The position is ___.5, so I averaged positions ___ and ___.

Mode: The mode is ___ because it appears ___ times.

Warm-Up: Mini Match Goals

Use this quick practice before research. Sort first, then use (N + 1) ÷ 2.

Data set

A team scored these goals in 8 matches:

2, 1, 3, 2, 4, 1, 2, 5

Check answer

Ordered: 1, 1, 2, 2, 2, 3, 4, 5. N = 8. (8 + 1) ÷ 2 = 4.5, so average the 4th and 5th values. Both are 2. Median = 2. Mode = 2.
