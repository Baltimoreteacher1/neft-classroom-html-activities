Notes + Student Practice

Median is the middle value after numbers are sorted from least to greatest. Mode is the value that appears most often.

Use (N + 1) ÷ 2 to find the median position. You do the work; the checks only tell you if you are on track.

Median Steps

  1. Organize from least to greatest.
  2. Count the values. This is N.
  3. Use (N + 1) ÷ 2.
  4. Whole number position: use that value.
  5. .5 position: average the two middle values.

Mode Steps

  1. Look for repeats.
  2. Count how many times each value appears.
  3. The most repeated value is the mode.
  4. If nothing repeats, write no mode.

Helpful Frames
