Warm-Up 1
What is the distance between the points (2, 3) and (2, −4) on the coordinate plane?
Vocabulary: distance is how far apart two points are (always positive). When two points share the same x-coordinate, find the distance by subtracting the y-coordinates and taking the absolute value.
Formula reference: |3 − (−4)| = |3 + 4| = 7
✓Correct! The points share x = 2, so distance = |3 − (−4)| = |3 + 4| = 7 units.
✗Not quite. Both points have x = 2, so find the vertical distance: |3 − (−4)| = |7| = 7 units (answer C).
Warm-Up 2
True or False: To find the distance between two points on the same horizontal line, you subtract the x-coordinates and take the absolute value.
Vocabulary: A horizontal line goes left-right. Points on the same horizontal line share the same y-coordinate.
Absolute value makes any number positive, so distance is always positive.
✓Correct! On a horizontal line the y-coordinates are the same, so you subtract the x-coordinates and use absolute value to get the distance.
✗Actually, the statement is true. Points on the same horizontal line share a y-coordinate, so distance = |x₂ − x₁|.
