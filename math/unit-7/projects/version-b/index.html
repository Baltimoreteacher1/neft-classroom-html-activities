Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

Absolute Value

|n| is the distance from 0 on a number line — never negative. |−120| = 120 m from sea level.

Valor absoluto: |n| es la distancia desde 0 — nunca es negativo. |−120| = 120.

Ordering Integers

On a number line, left is less. More negative = smaller. −7 < −3 < 0 < 2.

Ordenar enteros: a la izquierda es menor. Más negativo = más pequeño.

Sea Level = 0 (reference)

Zero is the reference point. Above sea level is positive (+), below is negative (−).

Nivel del mar = 0: arriba es positivo (+), abajo es negativo (−).

Integer (positive / negative)

Whole numbers including negatives: …−2, −1, 0, 1, 2… Negatives describe below-zero amounts.

Entero: números enteros con negativos: …−2, −1, 0, 1, 2…

Range / Spread (Level 2)

Range = greatest − least. It measures how spread out a set of readings is.

Rango = mayor − menor. Mide qué tan dispersos están los datos.

Formula Bank

Absolute value: |n| = distance from 0 (sea level = 0)

Distance from sea level: |depth|

Total vertical span: |lowest| + |highest|

Order: least → greatest, left → right on the number line

Safe check: |depth| ≤ limit → safe

Range (Level 2): greatest − least

Solving One-Step Equations:

  1. Identify the operation happening to the variable (addition, subtraction, multiplication, or division).
  2. Apply the **Inverse (Opposite) Operation** to both sides of the equals sign to isolate the variable:
    • Opposite of addition ($+$) is subtraction ($-$).
    • Opposite of subtraction ($-$) is addition ($+$).
    • Opposite of multiplication ($\times$) is division ($÷$).
    • Opposite of division ($÷$) is multiplication ($\times$).
  3. Solve to find the value of the variable.

Part 1b · Self — your real plan

📏 Distance from sea level (absolute value)

The absolute value of your lowest place tells how far below sea level it is — always positive.

Rule: distance from sea level = |depth|. A depth of −120 m means 120 m below sea level. |−120| = 120.
Need a hint?

Absolute value is the distance from zero — never negative. |−45| = 45 because −45 is 45 units left of 0. Remove the sign, keep the number.

🌡️ Order your temperature life (compare & order)

Enter real temperatures from your life. Sort them from coldest to warmest. More negative = colder.

Need a hint?

On a number line, numbers to the left are less. −18 < −5 < 4 < 18 < 35. The most negative number is the coldest.

Level 2 — Range & Outlier Detective: after sorting, the page reports your range (warmest − coldest) and flags any value far from the others.
