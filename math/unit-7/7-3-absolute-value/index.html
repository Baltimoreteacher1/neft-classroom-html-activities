Practice 4
True or false: "|−4| > |3| means −4 > 3."
Be careful! Comparing absolute values is NOT the same as comparing the numbers themselves.
|−4| = 4 and |3| = 3, so 4 > 3 is true.
But is −4 > 3? Where is −4 on the number line compared to 3?
Explain your reasoning:

Sentence frame: "|−4| = ___ and |3| = ___, so |−4| > |3| is ___. But −4 > 3 is ___ because −4 is to the ___ of 3 on the number line."

✓Correct! |−4| > |3| is true (4 > 3), but −4 is NOT greater than 3. Comparing absolute values is different from comparing the numbers.
✗This is false. While |−4| > |3| (because 4 > 3), that does not mean −4 > 3. In fact, −4 < 3. Absolute value comparison does not equal number comparison.
