Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

⚖️Mean (average)

Add all the values, then divide by how many there are.

Media (promedio): suma todos los valores y divide entre cuántos hay.

🎯Median

Put the numbers in order; the middle value is the median.

Mediana: ordena los números; el valor del medio es la mediana.

🔁Mode

The value that appears most often.

Moda: el valor que aparece más veces.

📏Range

How spread out the data is: largest − smallest.

Rango: qué tan disperso está: mayor − menor.

📐MAD

Mean Absolute Deviation: the average distance each value is from the mean.

DAM (desviación absoluta media): la distancia promedio de cada valor a la media.

📈Distribution shape

The overall pattern: symmetric, skewed left/right, or has outliers.

Forma de la distribución: el patrón general: simétrica, sesgada o con valores atípicos.

Formula Bank

Mean = sum of values ÷ count

Median = order the data → middle value

MAD = average of |value − mean|

Range = maximum − minimum

Media = suma ÷ cantidad · Mediana = ordenar → medio · DAM = promedio de |valor − media| · Rango = máximo − mínimo

Statistical Questions · 6.SP.1

Is It a Statistical Question?

A statistical question is one you would collect data to answer — and the data has variability (different people or things give different answers). A non-statistical question has one definite answer.

Directions: For each question below, decide whether it is statistical (S) or non-statistical (NS). Click your choice, then hit Check to see if you are right.

1. "How many minutes does each student in our class spend reading each day?"

2. "How many pages are in our math textbook?"

3. "What are the heights (in inches) of students on the school basketball team?"

4. "How many days are in a week?"

Need a hint?

Ask yourself: "Could different people, objects, or times give different answers?" If yes, the data would vary — so it is a statistical question. If there is one fixed answer (like 7 days in a week), it is non-statistical.

Sorted-data check: Sort your data least to greatest in your head, then type the middle value (the median).
Need a hint?

Mean: Add all values, then divide by how many there are.
Median: Sort the values from least to greatest. The middle value is the median. If there is an even count, average the two middle values.
Mode: The value that appears most often. A data set can have no mode, one mode, or multiple modes.

Measures of Variability · 6.SP.5c

Range and Mean Absolute Deviation

Using the same data set from Phase 2, compute the range (how spread out) and MAD (mean absolute deviation — the average distance each value is from the mean). Show your work.

Quick check: A data set is 4, 8, 6, 10, 2. What is the range?
Need a hint?

Range = Maximum value − Minimum value.
MAD steps: (1) Find the mean. (2) Subtract the mean from each value and take the absolute value — that is the deviation. (3) Find the average of all those absolute deviations. That average is the MAD.

Data Display & Shape · 6.SP.4 · 6.SP.2

Display and Describe the Distribution

Click Draw Dot Plot to see your data set visualized. Then choose the shape that best describes your distribution and check a quick-check problem.

Quick check: The data set is 3, 5, 7, 9, 11. What is the median?
Need a hint?

Sort the values: 3, 5, 7, 9, 11. There are 5 values, so the middle one (position 3) is the median.
Distribution shapes: Symmetric means the data is balanced on both sides of the center. Skewed right means a long tail stretches to the right. Skewed left means the tail goes left. Outliers are values much higher or lower than the rest.
