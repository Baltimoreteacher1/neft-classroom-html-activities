For the Teacher

How to Use These Projects

Standards: 6.RP.2 (unit rates), 6.RP.3b (unit price / better buy), 6.RP.3c (percent of a number, discounts, tax, fraction-decimal-percent conversions), 6.RP.3d (measurement unit conversion using rate reasoning).

Both versions are parallel. They assess the same rate and percent skills through different real-world contexts, so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: a Self → Peer → World arc, Visual Math Notes vocabulary, a Formula Bank and Sentence Frames, an EN/ES toggle, Level 1 / Level 2 differentiation, live calculators with self-checks, peer-compare and world-data tools with a What-If Lab, save/load and report export (.txt/.csv), a sticky progress bar, hints, a written deliverable, a student checklist, a rubric, and a print button.
