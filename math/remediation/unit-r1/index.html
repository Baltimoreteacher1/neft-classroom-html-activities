For the Teacher

Unit R1 Teacher Notes

Prerequisite: Students should be able to read whole numbers up to at least 10,000 before starting Lesson 2. Lesson 1 provides that foundation if needed.

Pacing: Each lesson is designed for a 20–30 minute independent session. All five lessons together make a solid one-week unit. The exit ticket in each lesson is a quick formative check — collect student responses before moving on.

Standards alignment: Lessons 1–4 directly address 6.NS.2 (fluently divide multi-digit numbers using the standard algorithm) by rebuilding prerequisite whole-number skills. Lesson 5 bridges to 6.EE.1/2 (write and evaluate numerical expressions involving whole-number exponents).

Pairs with: the Manipulatives Toolkit for hands-on modeling and the Bilingual Vocab Deck for ESOL support.
