Add & Subtract Decimals

Line up the decimal points, then add or subtract just like whole numbers. Use this every time you work with money.

Bridge to Grade 6 Unit R3 · Lesson 2 Builds 6.NS.3
Lesson progress: 0%
Why this matters for Grade 6: Grade 6 requires fluency adding and subtracting multi-digit decimals (6.NS.3). Every time you split a bill, calculate change, or add prices, you are using this skill. Getting the decimal point aligned is the one rule that makes it all work.
1
Learn It

The One Rule: Line Up the Decimal Point

When you add or subtract decimals, line up the decimal points first. Then add zeros if needed so every number has the same number of decimal places, and calculate column by column.

Example 1 — Add: $2.50 + $3.75
  2.50
+ 3.75
————
  6.25
Tenths: 5 + 7 = 12 → write 2, carry 1. Ones: 2 + 3 + 1 carry = 6. Answer: $6.25
Example 2 — Subtract (make change): $10.00 − $4.65
  10.00
− 4.65
————
   5.35
Borrow as needed. Answer: $5.35 in change.

Try It · Guided

Work These Together

1. You buy a sandwich for $3.40 and a drink for $1.25. How much do you spend in total? (Type the number, e.g. 4.65)

Hint

Line up: 3.40 + 1.25. Hundredths: 0 + 5 = 5. Tenths: 4 + 2 = 6. Ones: 3 + 1 = 4. Total = 4.65.

2. You pay $10.00 for a book that costs $6.75. How much change do you get back?

Hint

10.00 − 6.75. Borrow to get: 10.00 − 6.75 = 3.25.

10.00 − 4.65 = 5.35.

5. Which symbol makes it true?   (1.8 + 0.45) ___ 2.3

Hint

1.80 + 0.45: hundredths 0 + 5 = 5, tenths 8 + 4 = 12 → write 2, carry 1, ones 1 + 0 + 1 = 2. Result = 2.25. Compare 2.25 to 2.30: the tenths are equal (2), but hundredths 2 < 3, so 2.25 < 2.30.

6. Three friends split the cost of supplies. Anya spends $4.50, Ben spends $3.25, and Cara spends $2.10. How much did they spend altogether?

Hint

4.50 + 3.25 = 7.75. Then 7.75 + 2.10 = 9.85.

Hint

20.00 − 8.47. Borrow as needed: 20.00 − 8.47 = 11.53.
