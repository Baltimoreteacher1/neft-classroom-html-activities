The Coordinate Plane & Integers

Part 1: Plotting points in Quadrant I

A coordinate plane has two number-line axes. To plot a point (x, y): go across (right) x units on the x-axis, then go up y units on the y-axis. Always: across first, then up.

Example — Plot (3, 2): Go 3 right, then 2 up.
Step 1: Start at origin (0,0). Step 2: Move 3 right (x = 3). Step 3: Move 2 up (y = 2). Mark the point.

Part 2: Integers & the Number Line

Integers include positive whole numbers, negative whole numbers, and zero: … −3, −2, −1, 0, 1, 2, 3 …
Negative numbers are to the left of zero on a number line. The further right, the larger the number.

Integer Number Line:
Numbers to the right are greater. So −3 < 2 because −3 is to the left of 2.

Absolute value is the distance from zero on the number line — always a positive number (or zero).

|−4| = 4    |4| = 4    |0| = 0

Both −4 and 4 are 4 steps from zero, so they have the same absolute value.

1. To plot point (5, 3) on a coordinate grid, how far right do you move first?

Hint

For (x, y): x comes first — go across (right) by x, then up by y. (5, 3) → 5 right, 3 up.

Hint

Absolute value = distance from zero. −7 is 7 steps from zero, so |−7| = 7.

Practice · On Your Own

You Try

3. Which symbol makes this true?   −3 ___ 2

Hint

On a number line, −3 is to the left of 2. Numbers to the left are smaller, so −3 < 2.

4. What is |−9|?

Hint

Absolute value is distance from zero. −9 is 9 steps from zero, so |−9| = 9.

5. A point is plotted by moving 4 units right and 6 units up from the origin. What is its ordered pair?

Hint

Across first (x), then up (y). 4 right → x = 4. 6 up → y = 6. Ordered pair: (4, 6).

6. Which integer is farthest from zero: −6, 4, or −2?

Hint

|−6| = 6, |4| = 4, |−2| = 2. The largest absolute value is 6, so −6 is farthest from zero.

7. On the coordinate grid below, a point is at (2, 5). What are its coordinates? (Enter the x-value — how far right from the origin.)

The point is 2 units to the right of the origin, so x = 2. The full ordered pair is (2, 5).
