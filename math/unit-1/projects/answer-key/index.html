Both Versions  ·  New Self → Peer → World Structure

💬 Guidance for Part 2 (Peer), Part 3 (World) & Level 2

Both projects now follow a Self → Peer → World arc and use the student's own real numbers in Part 1, so Part 1 answers are student-specific. The worked solutions above use the built-in "Use example numbers" values, which match the old defaults exactly, so they remain valid for any student who clicks that button.

P1 Part 1 — Self (student-specific)

Grade the method, not a fixed number. Each tool recomputes live (oninput). A correct response shows: GCF/LCM via factors or the Euclid method, decimal sums lined up by place value, and division checked by multiplying back. The "Use example numbers" button reproduces the worked answers above (12 bags / LCM 24 / $275.45 spent / $7.65 per guest for A; $150.15 subtotal / $48.00 per teammate / 18 kits / $425.45 total for B).

P2 Part 2 — Peer (interview & compare)

Students type a partner's results, and the page auto-renders me / partner / difference cards. There is no single right answer; check that the difference is computed correctly and the reflection cites real numbers.

Worked example (Version A): If my cost per guest is $7.65 and my partner's is $9.20, the difference card reads $1.55 (|7.65 − 9.20|). A strong reflection: "My cost per guest is $1.55 lower because I had more guests sharing the same total."
Worked example (Version B): Subtotal $150.15 vs partner $182.40 → difference $32.25; kits 18 vs 12 → difference 6.

P3 Part 3 — World (reference datasets)

The world meter plots the student's value against the average of a built-in sample. Confirm the student reads "above / below / equal to the average" correctly and the What-If Lab shows the average shifting toward the added value.

L2 Level 2 — Distributive Property Extension (6.NS.4)

Level 2 asks students to rewrite their two supply totals as GCF × (a + b) using the distributive property.

Sample Level 2 Answers

Version A (stickers 48, bars 36): GCF = 12
48 + 36 = (12 × 4) + (12 × 3) = 12 × (4 + 3) = 12 × 7 = 84 total items

Version B (screws 72, bolts 54): GCF = 18
72 + 54 = (18 × 4) + (18 × 3) = 18 × (4 + 3) = 18 × 7 = 126 total items

Level 2 world questions: Version A — $18.50 × 40 guests = $740.00. Version B — $279.00 ÷ 6 students = $46.50 each.
