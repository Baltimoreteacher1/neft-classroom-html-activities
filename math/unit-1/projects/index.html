For the Teacher

How to Use These Projects

Standards: 6.NS.2 (multi-digit division), 6.NS.3 (decimal operations), 6.NS.4 (GCF, LCM, distributive property).

Both versions are parallel. They assess the same skills through different contexts, so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: live calculators with instant feedback, a progress bar, hints for scaffolding, a written deliverable, a student checklist, a 3-point rubric, and a print button so students can turn in a finished plan.
