For the Teacher

How to Use These Projects

Standards: 6.NS.2 (multi-digit division), 6.NS.3 (decimal operations), 6.NS.4 (GCF, LCM, distributive property).

Both versions are parallel. They assess the same skills through different contexts, so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: a Self → Peer → World arc (own real data → partner interview → real-world comparison), Visual Math Notes vocab with an EN/ES toggle, Formula Bank and Sentence Frames, live oninput calculators with instant feedback, a What-If Lab, Level 1 (support) / Level 2 (challenge, distributive-property extension) toggle, save/load, a report with Copy/.txt/.csv export, a progress bar, hints, a written deliverable, a checklist, a rubric, and print.
