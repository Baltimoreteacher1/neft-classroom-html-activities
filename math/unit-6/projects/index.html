For the Teacher

How to Use These Projects

Standards: 6.EE.1 (whole-number exponents), 6.EE.2 (write, read, and evaluate algebraic expressions), 6.EE.3 (apply properties of operations including the distributive property), 6.EE.4 (identify equivalent expressions).

Both versions are parallel. They assess the same skills through different contexts — game design vs. real-world pricing — so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: a Self → Peer → World arc with a Phase 0 personal setup (students design their own game / model an app they actually use), Visual Math Notes vocab with ESOL glosses, a Formula Bank and Sentence Frames, an EN/ES toggle, Level 1 (support) and Level 2 (challenge) tiers, live oninput calculators with worked steps, a peer formula-trade with crossover, a world meter, a live equivalent-expression prover, a What-If lab, save/load, a generated report with copy / .txt / .csv export, a sticky progress bar, hints, a checklist, a 4-point rubric, and a print button.
