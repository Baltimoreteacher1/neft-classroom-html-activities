Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

🔢Base

The number being multiplied in a power. In 3⁴ the base is 3.

Base: el número que se multiplica en una potencia. En 3⁴ la base es 3.

⬆️Exponent

How many times to use the base as a factor. 3⁴ uses 3 four times.

3333

Exponente: cuántas veces se usa la base como factor. 3⁴ usa el 3 cuatro veces.

🧩Expanded Form

Writing a power as repeated multiplication: 3⁴ = 3 × 3 × 3 × 3.

Forma desarrollada: escribir una potencia como multiplicación repetida: 3⁴ = 3 × 3 × 3 × 3.

✖️Coefficient

The number multiplied by a variable. In 3c the coefficient is 3.

Coeficiente: el número que multiplica a una variable. En 3c el coeficiente es 3.

🔤Variable

A letter that stands for a number, like c for coins.

Variable: una letra que representa un número, como c para monedas.

📌Constant

A fixed number that does not change. In 3c + 50, 50 is a constant.

Constante: un número fijo que no cambia. En 3c + 50, el 50 es constante.

🧱Term

A part of an expression separated by + or −. 3c + 50 has two terms.

Término: una parte de una expresión separada por + o −. 3c + 50 tiene dos términos.

🟰Equivalent

Two expressions that always give the same value for every input.

Equivalente: dos expresiones que siempre dan el mismo valor para cada entrada.

📤Distributive Property

Multiply outside the parentheses by each term inside: a(b + c) = ab + ac.

Propiedad distributiva: multiplica el factor de afuera por cada término de adentro: a(b + c) = ab + ac.

⬆️ Level-up score (exponents · 6.EE.1)

Level 1 tip: start with a small base like 2 or 3 and an exponent of 2 or 3.

Need a hint?

An exponent means repeated multiplication. 3⁴ = 3 × 3 × 3 × 3. Multiply the base by itself as many times as the exponent says — and count the factors. A common mistake is 3 × 4 = 12 instead of 3⁴ = 81.

✖️ Coin bonus expression (write & evaluate · 6.EE.2)

Set your own coin rate r (points per coin) and flat bonus k. Your expression is r·c + k. Then pick how many coins c a player collects.

Level 1 — pick the right expression: "3 points for each coin, plus a 50-point bonus" is written as…
Level 2 — name the parts: In your expression rc + k, write a sentence that names the coefficient, the variable, and the constant using your own numbers.
Need a hint?

"r points for each coin" means r times c, written rc. "Plus a flat bonus k" means + k. Put them together: rc + k. The coefficient is the rate r; the constant is the flat bonus k; the variable is c.

🟰 Equivalent Expression Prover (6.EE.4)

Are 2(c + 3) and 2c + 6 equivalent? Test three values of c — if they match every time, they are equivalent. The distributive property is the reason: 2(c + 3) = 2c + 6.
