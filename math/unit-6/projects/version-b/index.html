Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

🔢Base

The number being multiplied in a power. In 3⁴ the base is 3.

Base: el número que se multiplica en una potencia. En 3⁴ la base es 3.

⬆️Exponent

How many times to use the base as a factor. 3⁴ uses 3 four times.

Exponente: cuántas veces se usa la base como factor. 3⁴ usa el 3 cuatro veces.

🧩Expanded Form

Writing a power as repeated multiplication: 3⁴ = 3 × 3 × 3 × 3.

Forma desarrollada: escribir una potencia como multiplicación repetida: 3⁴ = 3 × 3 × 3 × 3.

✖️Coefficient

The number multiplied by a variable. In 9.99n the coefficient is 9.99.

Coeficiente: el número que multiplica a una variable. En 9.99n el coeficiente es 9.99.

🔤Variable

A letter that stands for a number, like n for months.

Variable: una letra que representa un número, como n para meses.

📌Constant

A fixed number that does not change. In 9.99n + 4.50, 4.50 is constant.

Constante: un número fijo que no cambia. En 9.99n + 4.50, el 4.50 es constante.

🧱Term

A part of an expression separated by + or −. 9.99n + 4.50 has two terms.

Término: una parte de una expresión separada por + o −. 9.99n + 4.50 tiene dos términos.

🟰Equivalent

Two expressions that always give the same value for every input.

Equivalente: dos expresiones que siempre dan el mismo valor para cada entrada.

📤Distributive Property

Multiply outside the parentheses by each term inside: a(b + c) = ab + ac.

Propiedad distributiva: multiplica el factor de afuera por cada término de adentro: a(b + c) = ab + ac.

Formula Bank

App cost: p · n + fee → price × months + setup fee

Evaluate: substitute n, multiply first, then add the fee

Equivalent (6.EE.4): same value for every n

Distributive (6.EE.3): a(b + c) = a·b + a·c

Crossover: where two plans cost the same

"My app ___ costs $___ per month plus a $___ setup fee."

"After ___ months my total is $___, because p·n + fee = ___."

"My plan is cheaper / pricier than my partner's because ___."

"Mi app ___ cuesta $___ por mes más una cuota inicial de $___."

🟰 Equivalent Expression Prover (6.EE.4)

A family plan is advertised two ways: 2(n + 3) and 2n + 6. Test three values of n — if they match every time, they are equivalent. The distributive property is the reason: 2(n + 3) = 2n + 6.
