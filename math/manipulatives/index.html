Math Manipulatives Toolkit

Five interactive tools that make abstract math concrete — fraction bars you can shade, a number line you can place points on, algebra tiles you can arrange, an area decomposer for composite shapes, and a coordinate plotter for all four quadrants. Open any tool full-screen for classroom projection.

Fractions · 6.NS.1

Fraction Bars

Pick a denominator, shade parts by clicking, and instantly see the fraction, decimal, and percent. Add a second bar to compare two fractions with automatic >, <, = judgment.

Open Fraction Bars → Number Sense · 6.NS.5–7

Number Line

Configure the range, place points by clicking or typing, and show absolute value as a distance from zero. Plot two points to measure the distance between them.

Open Number Line → Expressions · 6.EE.3–4

Algebra Tiles

Add x-tiles and unit tiles (positive or negative), see the running expression, combine like terms automatically, and explore the distributive property a(x + b).

Open Algebra Tiles → Geometry · 6.G.1

Area Decomposer

Set dimensions of two shapes — rectangles or a rectangle + triangle — on a live SVG grid. The tool computes each part's area and the composite total, showing b×h and ½bh.

Open Area Decomposer → Coordinate Plane · 6.NS.6

Coordinate Plotter

Plot points on a four-quadrant grid by typing coordinates or clicking. Identify the quadrant, reflect across either axis, and compute distances between aligned points.

For the Teacher

Projection & Use Tips

Full-screen projection: Open any tool and press F11 (or use your browser's full-screen mode) for a clean display on a projector or interactive whiteboard.

No setup needed: Every tool is fully self-contained — no accounts, no downloads, no internet required after the page loads.

Standards alignment: Fraction Bars (6.NS.1) · Number Line (6.NS.5, 6.NS.6, 6.NS.7) · Algebra Tiles (6.EE.3, 6.EE.4) · Area Decomposer (6.G.1) · Coordinate Plotter (6.NS.6, 6.NS.8).
