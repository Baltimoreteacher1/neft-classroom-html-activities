Variable Voyage

Pilot the Substitution Machine across the galaxy. Drop the fuel value into every slot, then fire the operations in the right order to compute each expression.

6.EE.A.2 · Evaluate & Write Expressions
How the machine works: Each expression like 5 + 8 × n appears as parts on a track. Drag the fuel tile (for example n = 6) into every n slot. Then tap the operation nodes. Only the operation that goes next in order will fire — an out-of-order tap is blocked. Each one collapses to its answer until a single value is left.
