For the Teacher

How to Use These Projects

Standards: 6.EE.9 — use variables to represent two quantities in a real-world problem that change in relationship to one another; write an equation to express one quantity (dependent variable) in terms of the other quantity (independent variable). Analyze the relationship using graphs and tables.

Both versions are parallel. They assess the same skills through different contexts, so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: a Self → Peer → World arc, Visual Math Notes vocab with EN/ES toggle, a Formula Bank and Sentence Frames, Level 1 / Level 2 differentiation, live oninput calculators, a tolerance-checked show-your-work grid, a classmate-interview compare, built-in real benchmarks with a world meter and What-If Lab, progress bar, hints, rubric, save/load, report copy/download (.txt/.csv), and print. Both versions now include a dynamically generated SVG line graph (Version B graphs both plans on one grid).
