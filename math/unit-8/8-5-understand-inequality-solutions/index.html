Practice 2
True or false: The inequality x < 10 has exactly 5 solutions.
Key idea: An inequality like x < 10 has infinite solutions — not just whole numbers. Values like 9, 8.5, 0, −3, and 9.999 all work. There is no limit to how many numbers are less than 10.
Explain why the inequality does or does not have exactly 5 solutions:

Sentence frame: "The inequality x < 10 does not have exactly 5 solutions because ___. It has ___ solutions."

✓Correct! The inequality x < 10 has infinitely many solutions. Any number less than 10 — including fractions and decimals — is a solution.
✗An inequality like x < 10 does not have just 5 solutions. It has infinitely many solutions: 9, 8, 7.5, 0, −100, and so on. The answer is false.

How to solve: Use inverse operations, just like equations. To undo "+ 3", subtract 3 from both sides. To undo "× 2", divide both sides by 2. The inequality sign stays the same direction.
Boundary value: the number where the inequality changes from true to false.

b.  2n ≤ 12  →  n ≤

Practice 5
A roller coaster requires riders to be at least 48 inches tall. This can be written as h ≥ 48. Which statement best describes the solution set?
Vocabulary: The solution set is the collection of all values that make an inequality true. "At least 48" means 48 and anything greater.
Hint: Think about the boundary value (48). Is 48 itself included?
Explain what the solution set means in this situation:

Sentence frame: "The solution set h ≥ 48 means that any height that is ___ or ___ is allowed on the ride."

✓Correct! h ≥ 48 means all heights that are 48 inches or greater. The ≥ symbol includes the boundary value 48.
✗The symbol ≥ means "greater than or equal to," so 48 is included. The solution set is all heights that are 48 inches or greater. The answer is C.
