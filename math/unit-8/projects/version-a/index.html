Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

🔤Variable

A letter (like x) that stands for an unknown number.

Variable: una letra (como x) que representa un número desconocido.

🔁Inverse Operation

The operation that undoes another: + undoes −, × undoes ÷.

Operación inversa: la que deshace otra: + deshace −, × deshace ÷.

✖️Coefficient

The number multiplied by the variable, like the 4 in 4x.

Coeficiente: el número que multiplica a la variable, como el 4 en 4x.

🎯Solution

The value of the variable that makes the equation true.

Solución: el valor de la variable que hace verdadera la ecuación.

⚖️Equation

A math sentence saying two amounts are equal, with an = sign.

Ecuación: una oración matemática que dice que dos cantidades son iguales (=).

🔎Substitution

Putting your answer back into the equation to check it.

Sustitución: poner tu respuesta en la ecuación para comprobarla.

↔️Inequality

Compares amounts with >, <, ≥, or ≤ instead of =.

Desigualdad: compara cantidades con >, <, ≥ o ≤ en vez de =.

⚪Open / Closed Circle

Open circle = not included (> or <); closed = included (≥ or ≤).

Círculo abierto = no incluido (> o <); cerrado = incluido (≥ o ≤).

⬆️At Least / At Most

"At least" means ≥ (or more). "At most" means ≤ (or fewer).

"Al menos" = ≥ (o más). "Como máximo" = ≤ (o menos).

Formula Bank

Add/Subtract: x + p = q → x = q − p

Multiply/Divide: p·x = q → x = q ÷ p

Substitution check: put x back in; both sides must match.

Two-step lock: p·x + r = q → x = (q − r) ÷ p

Inequality direction: "at least" → x ≥ value (closed circle, shade right).

Sentence Frames

"My equation is ___ because the clue says ___."

"To solve, I used the inverse operation ___, so x = ___."

"I checked by substitution: ___ + ___ = ___, so it is correct."

"Mi ecuación es ___ porque la pista dice ___. Para resolver usé la operación inversa ___, así que x = ___."

🔒 Lock A — Addition clue (x + p = q)

Level 1 tip: keep numbers small, like p = 5 and q = 12, so x is easy.

Your clue: "When you add p to the secret combination, you get q." The combination is the solution x. Use q > p so x is positive.
Need a hint?

Addition is undone by subtraction. For x + 8 = 15, subtract 8 from both sides: x = 15 − 8 = 7. Verify: 7 + 8 = 15 ✓

🔒 Lock B — Subtraction clue (x − p = q)

Your clue: "When you take away p from the combination, you get q." So x − p = q, which means x = q + p.
Need a hint?

Subtraction is undone by addition. For x − 6 = 10, add 6 to both sides: x = 10 + 6 = 16. Verify: 16 − 6 = 10 ✓

🔒 Lock C — Multiplication clue (p·x = q)

Your clue: "The combination times p equals q." Pick values so q ÷ p has no remainder (whole-number combination). Example: p = 4, q = 28 → x = 7.
Need a hint?

Multiplication is undone by division. For 4x = 28, divide both sides by 4: x = 28 ÷ 4 = 7. Verify: 4 × 7 = 28 ✓. Make sure q ÷ p has no remainder!

🔒 Lock D — Division clue (x ÷ p = q)

Your clue: "The combination divided by p equals q." So x ÷ p = q, which means x = q × p.

🔒 Lock D — Two-step combo lock (p·x + r = q)

Level 2 challenge: Make a harder lock with two steps: p·x + r = q, so x = (q − r) ÷ p. Choose values so the combination is a whole number greater than 50 to make a tough final lock.
Need a hint?

Undo in reverse order: first subtract r from both sides, then divide by p. For 3x + 7 = 220, subtract 7 → 3x = 213, divide by 3 → x = 71. Verify: 3 × 71 + 7 = 220 ✓
