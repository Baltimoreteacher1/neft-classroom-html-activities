🗝️ Your Mission

An escape room company has hired you to design four puzzle locks. Each lock opens only when a player solves a one-step equation. Work through four phases: translate a clue into an equation, write and solve addition/subtraction equations, write and solve multiplication/division equations, and finally sequence your puzzles and verify a known answer. Fill every box, hit Solve & Verify, and finish the blueprint checklist. Then print your escape room answer key.

Writing Equations · 6.EE.6

Translate the Clue

Good puzzle designers always start with a story clue and write it as an equation. Use a variable for the unknown combination number.

Puzzle Clue (Lock A): "The secret combination is a number. When you add 7 to it, you get 20. What is the combination?"

Write this as an equation using x for the unknown combination.
Need a hint?

A variable is a letter (like x) that stands for the unknown. "A number plus 7 equals 20" → x + 7 = 20. Your equation must have a variable, an operation, an equal sign, and a constant on the right side.

Addition & Subtraction Equations · 6.EE.7

Solve Lock C (Add/Subtract)

Design a puzzle where a player must solve an addition or subtraction equation. The calculator below solves x + p = q (or x − p = q) and verifies by substitution — just like the escape room answer key.

Lock C — Addition Equation: Choose your own values for p and q. The equation is x + p = q, so x = q − p. The solution must be positive (use q > p).
Need a hint?

Use the inverse operation: addition is undone by subtraction. For x + 8 = 15, subtract 8 from both sides: x = 15 − 8 = 7. Then verify: 7 + 8 = 15 ✓

3
Multiplication & Division Equations · 6.EE.7

Solve Lock D (Multiply/Divide)

Now design a trickier puzzle using multiplication. The equation is p · x = q, so x = q ÷ p. Pick values so the answer is a whole number — that keeps the escape room fair.

Lock D — Multiplication Equation: Choose p and q so that q is evenly divisible by p (no remainders). Example: p = 4, q = 28 → x = 7.
Need a hint?

Use the inverse operation: multiplication is undone by division. For 4x = 28, divide both sides by 4: x = 28 ÷ 4 = 7. Then verify: 4 × 7 = 28 ✓. Make sure q ÷ p has no remainder!
