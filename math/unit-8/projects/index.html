For the Teacher

How to Use These Projects

Standards: 6.EE.5 (substitution to check solutions), 6.EE.6 (using variables to write expressions and equations), 6.EE.7 (solving one-step equations x + p = q and px = q with nonnegative rational solutions), 6.EE.8 (writing and graphing inequalities to represent real-world constraints).

Both versions are parallel. They assess the same skills through different contexts, so you can assign Version A and Version B to different students and grade on the same rubric (built into each project). Great for choice boards, make-up work, or A/B class sections.

Built in: a Self → Peer → World arc, Visual Math Notes vocabulary with an EN/ES toggle, a Formula Bank and Sentence Frames, Level 1 / Level 2 differentiation, live calculators with instant substitution verification, a peer "swap & crack" / "who's closer" comparison, a built-in world data set with a comparison meter, a What-If Lab, save/load, report export (.txt/.csv), a sticky progress bar, scaffolded hints, a checklist, a 4-point rubric, a print button, and SVG number-line diagrams.
