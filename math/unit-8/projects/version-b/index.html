Before you start · Vocabulary

Visual Math Notes

Learn these words first. Tap EN / ES help at the top for Spanish.

🔤Variable

A letter (like x) that stands for an unknown number.

Variable: una letra (como x) que representa un número desconocido.

🔁Inverse Operation

The operation that undoes another: + undoes −, × undoes ÷.

Operación inversa: la que deshace otra: + deshace −, × deshace ÷.

✖️Coefficient

The number multiplied by the variable, like price in price·n.

Coeficiente: el número que multiplica a la variable, como el precio en precio·n.

🎯Solution

The value that makes the equation true.

Solución: el valor que hace verdadera la ecuación.

⚖️Equation

A math sentence saying two amounts are equal (=).

Ecuación: dice que dos cantidades son iguales (=).

🔎Substitution

Putting your answer back in to check it.

Sustitución: poner tu respuesta en la ecuación para comprobarla.

↔️Inequality

Compares amounts with >, <, ≥, or ≤ instead of =.

Desigualdad: compara con >, <, ≥ o ≤ en vez de =.

🌒Solution Set

All the values that make an inequality true.

Conjunto solución: todos los valores que hacen verdadera la desigualdad.

⚪Open / Closed Circle

Open = not included (> or <); closed = included (≥ or ≤).

Abierto = no incluido (> o <); cerrado = incluido (≥ o ≤).

⬆️At Least / At Most

"At least" means ≥ (or more). "At most" means ≤ (or fewer).

"Al menos" = ≥ (o más). "Como máximo" = ≤ (o menos).

Formula Bank

How much more: raised + x = goal → x = goal − raised

Items to sell: price · n = target → n = target ÷ price

Goal condition: "at least the goal" → x ≥ goal (closed circle, shade right)

Compound inequality: goal ≤ x ≤ stretch goal (between two amounts)

Minimize items: fewest n with price · n ≥ amount needed → n = ⌈needed ÷ price⌉

Sentence Frames

"We have raised ___, so we still need ___ to reach our goal of ___."

"At ___ per item, we must sell ___ items to earn ___."

"Our plan ___ meet the goal because ___ ≥ / < ___."

"Hemos recaudado ___, así que aún necesitamos ___ para llegar a la meta de ___."

1b
Part 1b · Self — your real plan

🧮 Plan the sales (price · n = target)

Your sales plan: choose a price per item and a sales target. The target must be divisible by the price so the number of items is a whole number.
Need a hint?

The equation is price · n = target. Divide both sides by price. For 5n = 150 → n = 150 ÷ 5 = 30 items. Verify: 5 × 30 = 150 ✓.

1c
Part 1c · Self — your real plan

↔️ Set the goal condition (inequality)

Level 1: use ≥ ("at least the goal").

Condition: the total raised (x) must be at least your goal. Write x ≥ goal, then see the solution set and number line.
Need a hint?

For "at least $300", use ≥: x ≥ 300. The solution set is 300, 301, 310, 400 … On a number line draw a closed circle at 300 (includes 300) and shade right. Strictly > uses an open circle.

Level 2 — compound inequality: set a stretch goal so the "great" range is goal ≤ x ≤ stretch. Then minimize items: what is the fewest number of items at your price to at least cover the amount still needed?
