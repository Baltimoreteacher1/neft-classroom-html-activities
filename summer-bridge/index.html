Review for MCAP

Teachers and parents: there is no single right order. A good summer plan is to start with the Skills Checkup, do a few short Spiral Review sets each week to stay sharp, drop into the Bridge studio for any skill that needs more work, and use the MCAP review when you want a fuller check.
