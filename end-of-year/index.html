For teachers & families

How to run the final stretch

Suggested use: point students to one path at a time. Start with Quick Review or a short MCAP domain set to surface gaps, assign two or three targeted activities, then close with a full practice test or a project as evidence of learning.

Every link routes to an existing activity — nothing new to set up. Use the teacher dashboard to see results where activities report them, and the MCAP Review hub for domain breakdowns.
